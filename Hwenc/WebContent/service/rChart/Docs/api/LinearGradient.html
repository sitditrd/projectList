Class: LinearGradient

LinearGradient

new LinearGradient()

LinearGradient 클래스는 선형(Linear) 그라데이션 효과를 지원합니다.

LinearGradient 클래스 인스턴스의 entires 속성에 GradientEntry 클래스 인스턴스 배열을 정의해야 합니다.

게이지 차트에만 지원됩니다.

레이아웃은 다음과 같은 형식으로 작성됩니다.

<fill>
  <LinearGradient>
    <entries>
      <GradientEntry color="0xC5C551" ratio="0.00" alpha="0.5" />
      <GradientEntry color="0xFEFE24" ratio="0.33" alpha="0.5" />
      <GradientEntry color="0xECEC21" ratio="0.66" alpha="0.5" />
    </entries>
  </LinearGradient>
</fill>

Members

angleNumber

그라데이션의 진행 방향을 설정합니다.

기본값(default)은 0 으로 왼쪽에서 오른쪽으로 그라데이션이 진행됩니다.

값을 180 으로 설정하면 오른쪽에서 왼쪽으로 그라데이션이 진행됩니다.

Default Value:
  • 0

entriesObject

그라데이션 요소를 설정합니다.

유효값은 GradientEntry 클래스 인스턴스의 배열입니다.

idString

식별자(Identifier)를 설정합니다.

식별자(Identifier)는 사용자에 의해 정의된 임의의 고유값입니다.

설정된 식별자(Identifier)를 통해 해당 객체에 접근이 가능합니다.

Default Value:
  • null
