Class: GradientEntry

GradientEntry

new GradientEntry()

GradientEntry 클래스는 그라데이션 색상을 설정합니다.

LinearGradient, RadialGradient 클래스 인스턴스의 entries 속성값으로 설정됩니다.

정의된 GradientEntry 인스턴스의 수 만큼 그라데이션 효과가 처리됩니다.

게이지 차트에만 지원됩니다.

레이아웃은 다음과 같은 형식으로 작성됩니다.

<fill>
  <RadialGradient angle="45" focalPointRatio="-0.5">
    <entries>
      <GradientEntry color="0xC5C551" ratio="0.00" alpha="0.5" />
      <GradientEntry color="0xFEFE24" ratio="0.33" alpha="0.5" />
      <GradientEntry color="0xECEC21" ratio="0.66" alpha="0.5" />
    </entries>
  </RadialGradient>
</fill>

Members

alphaNumber

투명도를 설정합니다.

유효값은 0 과 1 사이의 값입니다.

Default Value:
  • 1

colorHexadecimal

색상을 설정합니다.

Default Value:
  • #000000

idString

식별자(Identifier)를 설정합니다.

식별자(Identifier)는 사용자에 의해 정의된 임의의 고유값입니다.

설정된 식별자(Identifier)를 통해 해당 객체에 접근이 가능합니다.

Default Value:
  • null

ratioNumber

전체 그라데이션에서 현재 요소가 차지하는 비율을 설정합니다.

유효값은 0 과 1 사이의 값입니다.

Default Value:
  • 0
