Class: ImageSourceItem

ImageSourceItem

new ImageSourceItem()

ImageSourceItem 클래스는 표현될 이미지에 대한 경로와 원본 이미지의 비율 등의 값을 설정합니다.

레이아웃은 다음과 같은 형식으로 작성됩니다.

 <ImageChart>
   ...
   <series>
     <ImageSeries ... >
       <imgSource>
         <ImageSourceItem maintainAspectRatio="true" url="../KoolChart/Assets/Images/man5.png" />
       </imgSource>
     </ImageSeries>
     <ImageSeries ... >
       <imgSource>
         <ImageSourceItem maintainAspectRatio="true" url="../KoolChart/Assets/Images/woman4.png" />
       </imgSource>
     </ImageSeries>
   </series>
 </ImageChart>

Members

maintainAspectRatioBoolean

표현되는 이미지에 대한 원본 비율을 유지할 지 여부를 설정합니다.

Default Value:
  • true

urlString

이미지의 경로를 설정합니다.

Default Value:
  • null

valueNumber

imageDisplayType 속성값이 multiple 일 경우 각 ImageSourceItem 인스턴스에 의해서 표현되는 데이터 값을 설정합니다.

Default Value:
  • NaN
