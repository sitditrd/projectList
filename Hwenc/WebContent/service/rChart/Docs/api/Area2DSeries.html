Class: Area2DSeries

Area2DSeries

new Area2DSeries()

Area2DSeries 클래스는 Area2DChart 클래스 인스턴스의 series 속성에 설정됩니다.

차트에 설정된 데이터를 처리하고 영역 차트를 표현합니다.

레이아웃은 다음과 같은 형식으로 작성됩니다.

<rMateChart>
  <Options>
    <Caption text="Annual Report"/>
  </Options>
  <Area2DChart showDataTips="true">
    <horizontalAxis>
      <CategoryAxis categoryField="Month" />
    </horizontalAxis>
    <series>
      <Area2DSeries yField="Profit" />
      <Area2DSeries yField="Cost" />
      <Area2DSeries yField="Revenue" />
    </series>
  </Area2DChart>
</rMateChart>
