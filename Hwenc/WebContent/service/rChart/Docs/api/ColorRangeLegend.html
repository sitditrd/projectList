Class: ColorRangeLegend

ColorRangeLegend

new ColorRangeLegend()

ColorRangeLegend 클래스는 ColorAxis 클래스 인스턴스의 색상값을 가지고 색상 범례를 표현합니다.

레이아웃은 다음과 같은 형식으로 작성됩니다.

...
<Options>
  <ColorRangeLegend axis="{colorAxis}" width="300" />
</Options>
...
<colorAxis>
  <ColorAxis id="colorAxis" maximum="400" minimum="0" interval="100" colors="[#fcd26b,#cc3635]" />
</colorAxis>
...
