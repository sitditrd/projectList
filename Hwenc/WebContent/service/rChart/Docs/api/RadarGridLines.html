Class: RadarGridLines

RadarGridLines

new RadarGridLines()

RadarGridLines 클래스는 레이더 차트의 배경에 그리드 선에 대한 표현을 지원합니다.

이 클래스를 사용하기 위해서는 레이아웃에서 backgroundElements 속성값으로 RadarGridLines 클래스 인스턴스를 설정해야 합니다.

레이아웃은 다음과 같은 형식으로 작성됩니다.

<rMateChart backgroundColor="0xFFFFFF" borderRadius="12" borderStyle="solid">
  ...
  <RadarChart id="chart1" isSeriesOnAxis="true" type="polygon" paddingTop="20" paddingBottom="20" showDataTips="true">
    ...
    <backgroundElements>
      <RadarGridLines direction='both' radarChangeCount='1' radarFill='0xEFFFFF' radarAlternateFill='0xFFFFFC'> <!--radarFill 과 radarAlternateFill 색이 차례로 나옵니다.-->
        <radialStroke>
          <Stroke color='0x000000' alpha='1' weight='1' />
        </radialStroke>
        <angularStroke>
          <Stroke color='0xFF0000' alpha='1' weight='1' />
        </angularStroke>
      </RadarGridLines>
    </backgroundElements>
    ...
  </RadarChart>
</rMateChart>

Extends

  • GridLines

Members

angularStrokeObject

레이더 차트 배경에 표시될 angular 선의 형태를 설정합니다.

유효값은 Stroke 클래스의 인스턴스입니다.

Default Value:
  • Stroke("#bfbfbf")

directionString

차트 배경선(gridLine)의 유형을 설정합니다.

유효값은 vertical(수직선), horizontal(수평선), both(수직선과 수평선 동시) 입니다.

Default Value:
  • horizontal

gridLinesString

레이더 차트의 배경에 표시할 선의 형태를 설정합니다.

유효값은 다음과 같습니다.


  • radial - 중심에서 AngularAxis 인스턴스의 축까지 직선을 표시합니다.

  • angular - AngularAxis 인스턴스의 축 라벨들을 서로 연결하는 선을 표시합니다. 이때 레이더 차트의 유형(type)이 circle 이면 원으로 선을 표시하고 polygon 이면 다각형으로 선을 표시합니다.

Default Value:
  • both

horizontalAlternateFillObject

direction 속성값이 horizontal 일 경우 다음 채우기 색상을 설정합니다.

이 속성에 설정된 색상이 horizontalFill 속성에 설정된 색상과 번갈아서 적용됩니다.

유효값은 SolidColor 클래스의 인스턴스입니다.

Default Value:
  • null

horizontalCenterStrokeObject

차트 배경의 중앙에 표시할 가로선의 스타일을 설정합니다.

유효값은 Stroke 클래스의 인스턴스입니다.

Default Value:
  • Stroke("#b1b1b1")

horizontalChangeCountNumber

horizontalFill 속성과 horizontalAlternateFill 속성이 서로 번갈아서 적용되는 횟수를 설정합니다.

Default Value:
  • 1

horizontalFillObject

direction 속성값이 horizontal 일 경우 시작 채우기 색상을 설정합니다.

유효값은 SolidColor 클래스의 인스턴스입니다.

Default Value:
  • null

horizontalOriginStrokeObject

direction 속성값이 horizontal 인 경우, 수직축 값이 0 이 되는 위치에서 그려지는 수평선의 색상을 설정합니다.

유효값은 Stroke 클래스의 인스턴스입니다.

Default Value:
  • Stroke("#555555")

Default Value:
  • false

이 속성에 설정된 색상이 radarFill 속성에 설정된 색상과 번갈아서 적용됩니다.
