Class: GridLines

GridLines

new GridLines()

GridLines 클래스는 차트에서 데이터가 표현되는 영역의 배경에 격자선을 표시합니다.

차트의 축상에 있는 모든 눈금(tick)에 격자선이 표시됩니다. 추가로 차트의 중앙에 세로선과 가로선을 표시할 수 있습니다.

레이아웃에서 backgroundElements 또는 annotationElements 속성값으로 GridLines 클래스 인스턴스를 설정합니다.

레이아웃은 다음과 같은 형식으로 작성됩니다.

<rMateChart backgroundColor="0xFFFFFF">
  <Bar3DChart>
    <verticalAxis>
      <CategoryAxis categoryField="Month" />
    </verticalAxis>
    <series>
      <Bar3DSeries xField="Profit" />
    </series>
    <backgroundElements>
      <GridLines direction="vertical" verticalChangeCount="1">
        <verticalStroke>
          <Stroke color="0x999999" alpha="0.5" weight="1" />
        </verticalStroke>
        <verticalFill>
          <SolidColor color="0xFFFFFF" alpha="0.5" />
        </verticalFill>
        <verticalAlternateFill>
          <SolidColor color="0xEEEEEE" alpha="0.5" />
        </verticalAlternateFill>
      </GridLines>
    </backgroundElements>
  </Bar3DChart>
</rMateChart>

Members

directionString

차트 배경선(gridLine)의 유형을 설정합니다.

유효값은 vertical(수직선), horizontal(수평선), both(수직선과 수평선 동시) 입니다.

Default Value:
  • horizontal

horizontalAlternateFillObject

direction 속성값이 horizontal 일 경우 다음 채우기 색상을 설정합니다.

이 속성에 설정된 색상이 horizontalFill 속성에 설정된 색상과 번갈아서 적용됩니다.

유효값은 SolidColor 클래스의 인스턴스입니다.

Default Value:
  • null

horizontalCenterStrokeObject

차트 배경의 중앙에 표시할 가로선의 스타일을 설정합니다.

유효값은 Stroke 클래스의 인스턴스입니다.

Default Value:
  • Stroke("#b1b1b1")

horizontalChangeCountNumber

horizontalFill 속성과 horizontalAlternateFill 속성이 서로 번갈아서 적용되는 횟수를 설정합니다.

Default Value:
  • 1

horizontalFillObject

direction 속성값이 horizontal 일 경우 시작 채우기 색상을 설정합니다.

유효값은 SolidColor 클래스의 인스턴스입니다.

Default Value:
  • null

horizontalOriginStrokeObject

direction 속성값이 horizontal 인 경우, 수직축 값이 0 이 되는 위치에서 그려지는 수평선의 색상을 설정합니다.

유효값은 Stroke 클래스의 인스턴스입니다.

Default Value:
  • Stroke("#555555")

horizontalShowCenterLineBoolean

차트 배경의 중앙에 가로선을 표시할 지 여부를 설정합니다.

Default Value:
  • false

horizontalShowOriginBoolean

수평축 원본선을 표시할 지 여부를 설정합니다.

Default Value:
  • false

horizontalStrokeObject

차트 배경 수평선의 색상을 설정합니다.

유효값은 Stroke 클래스의 인스턴스입니다.

Default Value:
  • Stroke("#f1f1f1")

horizontalTickAlignedBoolean

수평축을 눈금선(tickMark)과 동일 선상에 표시할 지 여부를 설정합니다.

Default Value:
  • true

showEdgeHorizontalStrokeBoolean

수평축의 처음과 끝에 선을 표시할 지 여부를 설정합니다.

Default Value:
  • true

showEdgeVerticalStrokeBoolean

수직축의 처음과 끝에 선을 표시할 지 여부를 설정합니다.

Default Value:
  • true
