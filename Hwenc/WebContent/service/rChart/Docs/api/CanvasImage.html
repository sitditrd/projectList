Class: CanvasImage

CanvasImage

new CanvasImage()

CanvasImage 클래스는 차트의 배경, Box 클래스 인스턴스, CanvasElement 클래스 인스턴스에 이미지의 표시를 지원합니다.

레이아웃은 다음과 같은 형식으로 작성됩니다.

...
<CanvasElement>
  <CanvasImage right="10" bottom="10" source="http://www.riamore.net/image/logo2.png" />
</CanvasElement>
...
