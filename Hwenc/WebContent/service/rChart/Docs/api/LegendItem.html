Class: LegendItem

LegendItem

new LegendItem()

LegendItem 클래스는 차트의 범례 항목들을 표현합니다.

마커(marker)와 문자열(라벨)을 출력합니다.

레이아웃은 다음과 같은 형식으로 작성됩니다.

<SubLegend direction="horizontal">
  <LegendItem label="More than 2,000" fontWeight="bold">
    <stroke>
      <Stroke weight="1" color="#000000" />
    </stroke>
    <fill>
      <SolidColor color="#FF3366" />
    </fill>
  </LegendItem>
</SubLegend>

Members

disableItemFillObject

범례 항목이 선택 해제되었을 경우 마커의 색상을 설정합니다.

fillObject

범례 항목 마커의 색상을 설정합니다.

horizontalGapNumber

마커(marker)와 문자열(라벨) 사이의 수평 여백의 크기를 설정합니다.

Default Value:
  • 8

itemRendererString

범례 항목에 대한 마커(marker)의 모양을 렌더링하는 클래스를 설정합니다.

labelString

출력하려는 문자열(라벨)을 설정합니다.

labelPlacementString

문자열(라벨)의 출력 위치를 설정합니다.

유효값 left, right 입니다.

Default Value:
  • right

markerHeightNumber

범례 항목에 대한 마커(marker)의 세로 크기를 설정합니다.

Default Value:
  • 9

markerWidthNumber

범례 항목에 대한 마커(marker)의 가로 크기를 설정합니다.

Default Value:
  • 9

paddingBottomNumber

하단 여백의 크기를 픽셀 단위로 설정합니다.

Default Value:
  • 0

paddingLeftNumber

왼쪽 여백의 크기를 픽셀 단위로 설정합니다.

Default Value:
  • 0

paddingRightNumber

오른쪽 여백의 크기를 픽셀 단위로 설정합니다.

Default Value:
  • 0

paddingTopNumber

상단 여백의 크기를 픽셀 단위로 설정합니다.

Default Value:
  • 3

verticalGapNumber

마커(marker)와 문자열(라벨) 사이의 수직 여백의 크기를 설정합니다.

Default Value:
  • 6
