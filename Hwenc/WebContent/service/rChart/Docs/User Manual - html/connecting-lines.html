데이터 연결선

박스 형태로 그래프가 표현되는 차트 종류(바 차트, 컬럼 차트, 윙 차트)에서 인접한 데이터 포인트를 연결하는 연결선을 그을 수 있습니다. 연결선을 긋기 위해서는 해당 차트의 시리즈 노드에 lineToEachItems 속성값을 “true” 로 설정합니다. 다음은 3D 컬럼 차트에서 연결선을 긋기 위한 코드와 이를 적용해서 출력한 차트의 예제입니다.

...
<Column3DSeries labelPosition="inside" yField="goods" displayName="goods" lineToEachItems="true" showValueLabels="[6]" color="#ffffff">
...
