Class: AngularAxisRenderer

AngularAxisRenderer

new AngularAxisRenderer()

AngularAxisRenderer 클래스는 레이더 차트에서 카테고리에 해당되는 축을 렌더링합니다.

축의 굵기, 축의 색, 눈금의 위치, 눈금의 색 등을 조절할 수 있습니다.

반드시 axis 속성에 해당 축을 설정해야 합니다.

레이아웃은 다음과 같은 형식으로 작성됩니다.

<RadarChart>
  <angularAxis>
    <CategoryAxis id="aAxis" categoryField="catName" displayName="Category">
  </angularAxis>
  <angularAxisRenderers>
    <AngularAxisRenderer axis="{aAxis}" styleName="hangingCategoryAxis">
  </angularAxisRenderers>
  ...
</RadarChart>

Members

axisObject

화면에 표시할 축의 객체 값(식별자)을 설정합니다.

Default Value:
  • null

axisStrokeObject

축을 표시하는 선의 속성을 설정합니다.

유효값은 Stroke 클래스의 인스턴스입니다.

Default Value:
  • null

axisTitleStyleNameString

축 타이틀의 스타일을 <Style> 노드에 설정할 경우, <Style> 노드에 정의된 이름을 설정합니다.

<Style> 노드에 스타일을 정의할 때는 반드시 스타일명 앞에 마침표가 위치해야 합니다.

레이아웃은 다음과 같은 형식으로 작성됩니다.

<angularAxisRenderers>
  <AngularAxisRenderer axis="{aAxis}" axisTitleStyleName="aStyle" ... />'
</angularAxisRenderers>'
...
<Style>
  .aStyle{color:#ffffff;fontSize:12;}
</Style>
Default Value:
  • null

idString

식별자(Identifier)를 설정합니다.

식별자(Identifier)는 사용자에 의해 정의된 임의의 고유값입니다.

설정된 식별자(Identifier)를 통해 해당 객체에 접근이 가능합니다.

Default Value:
  • null

labelGapNumber

라벨과 축 간의 간격을 설정합니다.

Default Value:
  • 6

showLabelsBoolean

라벨의 표시 여부를 설정합니다.

Default Value:
  • true

showLineBoolean

실제 축의 표시 여부를 설정합니다.

Default Value:
  • false
