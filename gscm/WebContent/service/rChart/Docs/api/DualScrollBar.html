Class: DualScrollBar

DualScrollBar

new DualScrollBar()

DualScrollBar 클래스는 DualChart 에 데이터가 많을 경우 dataSelector 속성에 클래스 인스턴스로 설정됩니다.

dataSelector 속성이 설정될 경우, dataSelector 속성값에 따른 데이터가 mainChart 와 subChart 에 표현됩니다.

레이아웃은 다음과 같은 형식으로 작성됩니다.

...
<mainChart>
  ...
</mainChart>
<subChart>
  ...
</subChart>
<dataSelector>
  <DualScrollBar visibleItemSize="30" inverted="false" />
</dataSelector>
...

Members

arrowScrollSizeNumber

스크롤바 화살표를 클릭하였을 때 이동하는 아이템의 개수를 설정합니다.

Default Value:
  • 1

arrowSizeNumber

스크롤바의 화살표가 위치하는 공간의 넓이를 설정합니다.

Default Value:
  • 13

invertedBoolean

차트에 설정된 데이터를 끝에서부터 표시할 지 여부를 설정합니다.

Default Value:
  • false

scrollHeightNumber

스크롤바의 높이를 설정합니다.

Default Value:
  • 13

thumbHeightNumber

스크롤바 썸(thumb)의 높이를 설정합니다.

Default Value:
  • 7

trackScrollSizeNumber

스크롤바 트랙을 클릭하였을 때 이동하는 아이템의 개수를 설정합니다.

Default Value:
  • visibleItemSize 속성값

visibleBoolean

스크롤바(DualScrollBar)를 화면에 표시할 지 여부를 설정합니다.

Default Value:
  • true

visibleItemSizeNumber

차트(mainChart, subChart)에 표시될 데이터의 개수를 설정합니다.

Default Value:
  • 10
