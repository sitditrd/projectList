Class: Box

Box

new Box()

Box 클래스는 Label 과 Image 클래스 인스턴스를 표시할 수 있도록 합니다.

horizontalAlign, verticalAlign 속성에 따라서 박스 내부에 표시될 요소들이 순서대로 표시됩니다.

레이아웃은 다음과 같은 형식으로 작성됩니다.

<Box horizontalAlign="left" verticalAlign="top">
  <Box direction="vertical">
    <Label text="Test"/>
    <Label text="Test2"/>
  </Box>
  <Box>
    <Column2DChart>
      ...
    </Column2DChart>
  </Box>
</Box>

Members

backgroundAlphaNumber

박스 배경색의 투명도를 설정합니다.

유효값은 0 과 1 사이의 값입니다.

backgroundColorHexadecimal

박스의 배경색을 설정합니다.

borderColorHexadecimal

박스 테두리 선의 색상을 설정합니다.

borderStyleString

박스 테두리 선의 스타일을 설정합니다.

유효값은 다음과 같습니다.


  • none - 테두리 선을 그리지 않도록 설정합니다.

  • solid - 테두리를 기준으로 테두리 자체에 선을 그리도록 설정합니다.

  • inset - 테두리를 기준으로 테두리 안쪽에 선을 그리도록 설정합니다.

  • outset - 테두리를 기준으로 테두리 바깥쪽에 선을 그리도록 설정합니다.

Default Value:
  • none

directionString

박스 내부에 표시될 요소들의 위치를 설정합니다.

유효값은 horizontal, vertical 입니다.

Default Value:
  • vertical

horizontalAlignString

수평 정렬에 대한 방식을 설정합니다.

유효값은 left, center, right 입니다.

Default Value:
  • left

horizontalGapNumber

수평 정렬시 박스 내부에 표시되는 요소들 사이의 간격의 크기를 설정합니다.

Default Value:
  • 6

horizontalScrollPolicyString

수평 방향의 스크롤 정책을 설정합니다.

유효값은 다음과 같습니다.


  • auto - 박스 내부에 표시될 요소들의 크기를 고려하여 스크롤 여부가 결정됩니다.

  • off - 박스 내부에 표시될 요소들의 크기와 상관없이 스크롤을 하지 않습니다.

  • on - 박스 내부에 표시될 요소들의 크기와 상관없이 스크롤을 합니다.

Default Value:
  • auto

paddingBottomNumber

하단 여백의 크기를 픽셀 단위로 설정합니다.

Default Value:
  • 0

paddingLeftNumber

왼쪽 여백의 크기를 픽셀 단위로 설정합니다.

Default Value:
  • 0

paddingRightNumber

오른쪽 여백의 크기를 픽셀 단위로 설정합니다.

Default Value:
  • 0

paddingTopNumber

상단 여백의 크기를 픽셀 단위로 설정합니다.

Default Value:
  • 0

verticalAlignString

수직 정렬에 대한 방식을 설정합니다.

유효값은 top, middle, bottom 입니다.

Default Value:
  • top

verticalGapNumber

수직 정렬시 박스 내부에 표시되는 요소들 사이의 간격의 크기를 설정합니다.

Default Value:
  • 6

verticalScrollPolicyString

수직 방향의 스크롤 정책을 설정합니다.

유효값은 다음과 같습니다.

  • off - 박스 내부에 표시될 요소들의 크기와 상관없이 스크롤을 하지 않습니다.

  • on - 박스 내부에 표시될 요소들의 크기와 상관없이 스크롤을 합니다.

Default Value:
  • auto
