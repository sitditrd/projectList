Class: HttpServiceRepeater

HttpServiceRepeater

new HttpServiceRepeater()

HttpServiceRepeater 클래스는 설정된 URL 로 RPC 요청을 수행합니다.

RPC 수행에 의해서 서버로부터 받은 데이터를 target 속성에 설정된 객체에 전달합니다.

이때 target 속성에 설정된 객체는 반드시 dataProvider 속성을 갖는 객체여야 합니다.

레이아웃은 다음과 같은 형식으로 작성됩니다.

...
<HttpServiceRepeater url="http://www.koolchart.com/realtimeSample/process3Data.php" target="{chart}" interval="3" method="get" />
...

Members

intervalNumber

RPC 요청 후 최대 대기 시간을 설정합니다.

Default Value:
  • 30

targetObject

RPC 수행에 의해서 서버로부터 받은 데이터를 전달할 객체를 설정합니다.

target 속성에 설정되는 객체는 반드시 dataProvider 속성을 갖는 객체여야 합니다.

urlString

RPC 요청를 위한 URL 을 설정합니다.

useFaultAlertBoolean

RPC 수행 실패 시 경고 메시지 창을 표시할 지 여부를 설정합니다.

Default Value:
  • true
