Class: DateFormatter

DateFormatter

new DateFormatter()

DateFormatter 클래스는 원하는 날짜의 형식으로 출력하는 것을 지원합니다.

차트에서 처리되는 기본 날짜 형식은 YYYY/MM/DD HH:NN:SS 입니다.

차트에서 처리되는 기본 날짜 형식이 아니면 일반 스트링으로 처리되기 때문에, DateFormatter 클래스를 적용하기 위해서는 데이터 형식이 기본 날짜 형식과 일치하도록 해야합니다.

레이아웃은 다음과 같은 형식으로 작성됩니다.

...
<DateFormatter id="dateFmt" formatString="D MMM, YY"/>
...
<DateTimeAxis ... formatter="{dateFmt}" ... />
...

Members

formatStringString

출력하고자 하는 날짜 형식을 설정합니다.

예를 들어, 날짜 형식을 "2009년 3월" 과 같이 표현하고자 하는 경우 formatStirng="YYYY년 M월" 이 됩니다.

적용되는 형식은 다음과 같습니다.


  • Y - 연(Year) (예) YY=05,YYYY=2005

  • M - 월(Month) (예) M=7,MM=07,MMM=Jul

  • D - 일(Day) (예) D=4,DD=05

  • H - 시(Hour) (예) H=1,HH=01

  • N - 분(Minute) (예) N=1,NN=01

  • S - 초(Second) (예) SS=30

Default Value:
  • null

idString

식별자(Identifier)를 설정합니다.

식별자(Identifier)는 사용자에 의해 정의된 임의의 고유값입니다.

설정된 식별자(Identifier)를 통해 해당 객체에 접근이 가능합니다.

Default Value:
  • null
