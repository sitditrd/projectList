Class: HistoryChart

HistoryChart

new HistoryChart()

HistoryChart 클래스는 디스플레이어(Displayer), 네비게이터(Navigator), 셀렉터(Selector)로 구성되어 있습니다.

레이아웃은 다음과 같은 형식으로 작성됩니다.

<rMateChart backgroundColor="0xFFFFFF" borderRadius="12" borderStyle="solid">
  <Options>
    <Caption text="History Chart" />
  </Options>
  <HistoryChart>
    <displayerChart>
      <Displayer id="chart1" showDataTips="true" width="100%" height="100%">
        <series>
          <Column2DSeries labelPosition="inside" id="columnSeries" maxColumnWidth="20" yField="Data1" ... />
        </series>
      </Displayer>
    </displayerChart>
    <navigator>
      <Navigator id="navi" width="100%" height="80">
        <series>
          <Area2DSeries name="A" yField="Data1" />
        </series>
      </Navigator>
    </navigator>
    <selector>
      <HistoryRangeSelector width="100%" startingRange="center" visibleItemSize="30" />
    </selector>
  </HistoryChart>
</rMateChart>

Members

displayerChartObject

History 차트에서 선택된 데이터를 보여주는 디스플레이어(Displayer)를 설정합니다.

유효값은 Displayer, Displayer3D 클래스 인스턴스입니다.

Default Value:
  • null

idString

식별자(Identifier)를 설정합니다.

식별자(Identifier)는 사용자에 의해 정의된 임의의 고유값입니다.

설정된 식별자(Identifier)를 통해 해당 객체에 접근이 가능합니다.

Default Value:
  • null

navigatorObject

History 차트에서 네비게이터(Navigator) 역할을 하는 차트를 설정합니다.

유효값은 Navigator 클래스 인스턴스입니다.

Default Value:
  • null

selectorObject

History 차트에서 셀렉터(Selector)를 설정합니다.

유효값은 HistoryRangeSelector 클래스 인스턴스입니다.

Default Value:
  • null

Methods

displayCompleteCallFunction(id)

차트의 생성이 완료되었을 때 실행될 자바스크립트 함수명을 설정합니다.

파라메터로는 차트에 설정된 식별자(Identifier)가 넘어갑니다.

<...Chart displayCompleteCallFunction="displayCompleteFunc" ...>
...
displayCompleteFunc(id) {
  ...
}
Name Type Description
id String

displayCompleteCallFunction 을 실행한 차트의 식별자(Identifier) 입니다.
