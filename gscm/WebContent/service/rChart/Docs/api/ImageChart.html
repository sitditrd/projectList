Class: ImageChart

ImageChart

new ImageChart()

ImageChart 클래스는 데이터 값을 이미지의 크기로 차트에 표현합니다.

ImageChart 의 series 속성에는 ImageSeries 클래스 인스턴스의 배열을 설정합니다.

이미지 차트에서 이미지 표현 유형은 다음 3가지가 있습니다.


  • single - 대표 이미지 한 개로 데이터 값을 표현합니다.

  • singleRepeat - 대표 이미지 한 개의 단위 값을 설정하여, 데이터 값에 이를 때까지 이미지를 쌓아올려 데이터 값을 표현합니다.

  • multiple - 다수의 이미지에 각각의 단위 값을 설정하여, 데이터 값을 표현합니다.

레이아웃은 다음과 같은 형식으로 작성됩니다.

<rMateChart backgroundColor="0xFFFFFF"  borderRadius="12" borderStyle="solid">
  <Options>
    <Caption text="Image Chart" />
  </Options>
  <ImageChart id="chart" showDataTips="true" gutterLeft="20" gutterRight="20"  showLabelVertically="true">
    <series>
      <ImageSeries yField="Data1" ... >
        <imgSource>
          <ImageSourceItem maintainAspectRatio="true" url="../Samples/Images/1-1-1.png" />
        </imgSource>
      </ImageSeries>
      <ImageSeries yField="Data2" ... >
        <imgSource>
          <ImageSourceItem maintainAspectRatio="true" url="../Samples/Images/1-1-2.png" />
        </imgSource>
      </ImageSeries>
    </series>
  </ImageChart>
</rMateChart>
