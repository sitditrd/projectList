Class: rMateChart

rMateChart

new rMateChart()

rMateChart 클래스는 알메이트 차트 레이아웃(계층 형태)에서 최상위 노드에 설정되는 클래스입니다.

생성되는 차트의 배경 및 전체적인 프레임을 설정합니다. 예를 들면, 차트의 배경색, 테두리 스타일, 여백 등을 조절할 수 있습니다.

레이아웃은 다음과 같은 형식으로 작성됩니다.

<rMateChart  paddingRight="10" paddingBottom="5" paddingTop="5" paddingLeft="10">
  <Options>
    <Caption text="Annual Report"/>
  </Options>
  ...
  ...
  ...
</rMateChart>

Members

backgroundAlphaNumber

차트 배경색의 투명도를 설정합니다.

유효값은 0 과 1 사이의 값입니다.

Default Value:
  • 1

backgroundColorHexadecimal

차트 배경색을 설정합니다.

Default Value:
  • #FFFFFF

borderColorHexadecimal

차트 테두리 선의 색상을 설정합니다.

Default Value:
  • #777777

borderRadiusNumber

차트 테두리 모서리선의 둥근 정도를 설정합니다.

Default Value:
  • 1

borderStyleString

차트 테두리 선의 스타일을 설정합니다.

유효값은 다음과 같습니다.


  • none - 테두리 선을 그리지 않도록 설정합니다.

  • solid - 테두리를 기준으로 테두리 자체에 선을 그리도록 설정합니다.

  • inset - 테두리를 기준으로 테두리 안쪽에 선을 그리도록 설정합니다.

  • outset - 테두리를 기준으로 테두리 바깥쪽에 선을 그리도록 설정합니다.

Default Value:
  • none

borderThicknessNumber

차트 테두리 선의 두께를 설정합니다.

borderStyle 속성값이 solid 인 경우만 적용됩니다

Default Value:
  • 1

paddingBottomNumber

차트 전체 영역의 하단 여백의 크기를 픽셀 단위로 설정합니다.

Default Value:
  • 0

paddingLeftNumber

차트 전체 영역의 왼쪽 여백의 크기를 픽셀 단위로 설정합니다.

Default Value:
  • 0

paddingRightNumber

차트 전체 영역의 오른쪽 여백의 크기를 픽셀 단위로 설정합니다.

Default Value:
  • 0

paddingTopNumber

차트 전체 영역의 상단 여백의 크기를 픽셀 단위로 설정합니다.

Default Value:
  • 0
