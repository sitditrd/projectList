Class: CanvasElement

CanvasElement

new CanvasElement()

CanvasElement 클래스는 Label 과 Image 클래스 인스턴스를 표현할 수 있습니다.

CanvasElement 클래스에 표시되는 요소들은 해당 요소에 설정된 X, Y 좌표에 따라서 표시됩니다.

레이아웃은 다음과 같은 형식으로 작성됩니다.

...
  <backgroundElements>
    <CanvasElement>
      <Label text="memo" x="10" y="10"/>
    </CanvasElement>
  </backgroundElements>
...
