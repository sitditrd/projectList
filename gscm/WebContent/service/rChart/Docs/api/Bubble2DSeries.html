Class: Bubble2DSeries

Bubble2DSeries

new Bubble2DSeries()

Bubble2DSeries 클래스는 Bubble2DChart 클래스 인스턴스의 series 속성에 설정됩니다.

차트에 설정된 데이터를 처리하고 X, Y 좌표에 버블(Bubble)을 표현합니다.

레이아웃은 다음과 같은 형식으로 작성됩니다.

<rMateChart>
  <Options>
    <Caption text="Annual Report" />
  </Options>
  <Bubble2DChart showDataTips="true">
    <horizontalAxis>
      <LinearAxis />
    </horizontalAxis>
    <vertical>
      <LinearAxis />
    </verticalAxis>
    <series>
      <Bubble2DSeries xField="xData1" yField="yData1" radiusField="zData1" />
      <Bubble2DSeries yField="xData2" yField="yData2" radiusField="zData2" />
    </series>
  </Bubble2DChart>
</rMateChart>

  1. X축은 숫자 축(LinearAxis)이어야 합니다.

  4. {"data":4},{"data":1},{"data":2},{"data":3} 순으로 데이터가 입력되었을 경우
