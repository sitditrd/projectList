Class: DualChart

DualChart

new DualChart()

DualChart 클래스는 주 차트(mainChart), 보조 차트(subChart), 선택자(dataSelector)로 구성됩니다.

레이아웃은 다음과 같은 형식으로 작성됩니다.

<rMateChart backgroundColor="0xFFFFFF" borderRadius="12" borderStyle="solid">
  <DualChart>
    <mainChart>
      <Candlestick2DChart id="chart1" showDataTips="true" width="100%" height="100%">
        <horizontalAxis>
          <CategoryAxis id="mainHAxis" categoryField="Date" />
        </horizontalAxis>
        <series>
          <Candlestick2DSeries openField="open" closeField="close" highField="high" lowField="low" displayName="Candle">
          </Candlestick2DSeries>
        </series>
      </Candlestick2DChart>
    </mainChart>
    <subChart>
      <Column2DChart id="sub">
        <horizontalAxis>
          <CategoryAxis categoryField="Date" id="naviHAxis"  padding="0" />
        </horizontalAxis>
        <verticalAxis>
          <LinearAxis id="vAxis" />
        </verticalAxis>
        <series>
          <Candle2DSeries name="A" yField="Data1" />
        </series>
      </Column2DChart>
    </subChart>
    <dataSelector>
      <DualScrollBar visibleItemSize="30" />
    </dataSelector>
  </DualChart>
</rMateChart>

Members

dataSelectorObject

데이터가 많을 경우 DualChart 에 스크롤을 생성합니다.

DualScrollBar 클래스 인스턴스를 설정할 수 있습니다.

leftGutterSyncEnableBoolean

주 차트(mainChart) 와 보조 차트(subChart) 의 왼쪽 축 여백을 동기화할 지 여부를 설정합니다.

Default Value:
  • false

mainChartObject

DualChart 에서 출력할 주(main) 차트를 설정합니다.

mouseWheelSensitivityNumber

마우스 휠 스크롤시 변경될 아이템의 개수를 설정합니다.

Default Value:
  • 3

rightGutterSyncEnableBoolean

주 차트(mainChart) 와 보조 차트(subChart) 의 오른쪽 축 여백을 동기화할 지 여부를 설정합니다.

Default Value:
  • false

subChartObject

DualChart 에서 출력할 보조(sub) 차트를 설정합니다.

subChartPositionString

보조 차트(subChart) 의 위치를 설정합니다.

유효값은 bottom, top 입니다.

Default Value:
  • bottom

Methods

displayCompleteCallFunction(id)

차트의 생성이 완료되었을 때 실행될 자바스크립트 함수명을 설정합니다.

파라메터로는 차트에 설정된 식별자(Identifier)가 넘어갑니다.

<...Chart displayCompleteCallFunction="displayCompleteFunc" ...>
...
displayCompleteFunc(id) {
  ...
}
Name Type Description
id String

displayCompleteCallFunction 을 실행한 차트의 식별자(Identifier) 입니다.
