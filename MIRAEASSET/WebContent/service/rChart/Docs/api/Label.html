Class: Label

Label

new Label()

Label 클래스는 Box 클래스 인스턴스에 텍스트의 표시를 지원합니다.

레이아웃은 다음과 같은 형식으로 작성됩니다.

...
<Box width="100%" direction="horizontal" fontSize="12" color="#555555" fontWeight="bold">
  <Label width="100%" height="19" text="Last Year" />
  <Label width="100%" height="19" text="This Year" />
</Box>
...
