Class: DataEditor

DataEditor

new DataEditor()

DataEditor 클래스는 차트의 데이터를 테이블 형태로 표현하고 데이터에 대한 수정을 지원합니다.

레이아웃은 다음과 같은 형식으로 작성됩니다.

<Options>
  <DataEditor showOnInit="true" formatter="{numFmt}" useCloseButton="false" editorHeight="94" reverseRow="true" textAlign="right" />
</Options>

Members

alternatingItemColorsArray

데이터 에디터(테이블 형태) 행 단위의 배경색(RGB)을 설정합니다.

반드시 2 개 이상의 값을 가지는 배열로 설정해야 합니다.

borderColorHexadecimal

테두리 선의 색상을 설정합니다.

borderStyleString

테두리 스타일을 설정합니다.

유효값은 다음과 같습니다.


  • none - 테두리 선을 그리지 않도록 설정합니다.

  • solid - 테두리를 기준으로 테두리 자체에 선을 그리도록 설정합니다.

  • inset - 테두리를 기준으로 테두리 안쪽에 선을 그리도록 설정합니다.

  • outset - 테두리를 기준으로 테두리 바깥쪽에 선을 그리도록 설정합니다.

Default Value:
  • solid

borderThicknessNumber

테두리 선의 두께를 설정합니다.

Default Value:
  • 1

editableBoolean

데이터 에디터의 필드에 대한 편집가능 여부를 설정합니다.

Default Value:
  • true

editorHeightNumber

데이터 에디터의 세로 크기를 픽셀 단위로 설정합니다.

Default Value:

반드시 2 개 이상의 값을 가지는 배열로 설정해야 합니다.

<Style> 노드에 스타일을 정의할 때는 반드시 스타일명 앞에 마침표가 위치해야 합니다.

유효값은 none, underline 입니다.

Default Value:
  • none

텍스트의 들여쓰기를 픽셀 단위로 설정합니다.

닫기 버튼에 대한 CSS 를 HTML 문서에 포함(include)해야 합니다.

rMateChartH5.css 또는 className - rMateChartH5__CloseButton 에 대한 CSS 를 설정합니다.

Default Value:
  • true

Default Value:
  • middle
