Class: ColorAxis

ColorAxis

new ColorAxis()

ColorAxis 클래스는 색상에 대한 범례를 표시할 필요가 있을 경우 사용됩니다.

레이아웃은 다음과 같은 형식으로 작성됩니다.

...
<Options>
  <ColorRangeLegend axis="{cAxis}/>
</Options>
<TreeMapChart>
  <colorAxis>
    <ColorAxis id="cAxis" minimum="0" maximum="100" colors="[#ffffff, #ff0000]" interval="20"/>
  </colorAxis>
</TreeMapChart>
...

Members

colorsArray

축에 색상값(Hexadecimal)들을 Array 로 설정합니다.

Default Value:
  • ["#ffffff", "#ff0000"]

intervalNumber

최소값과 최대값을 기준으로 interval 값의 간격을 두고 눈금을 생성합니다.

Default Value:
  • 10

maximumNumber

축의 최대값을 설정합니다.

Default Value:
  • 100

minimumNumber

축의 최소값을 설정합니다.

Default Value:
  • 0
