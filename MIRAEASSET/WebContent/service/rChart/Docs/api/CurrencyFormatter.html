Class: CurrencyFormatter

CurrencyFormatter

new CurrencyFormatter()

CurrencyFormatter 클래스는 일반적으로 통화 금액을 표현하는 목적으로 활용됩니다.

숫자 표현 형식에 대한 변경 및 숫사 앞, 뒤에 문자(열)의 삽입을 지원합니다.

예를 들어, 통화 단위 "$" 를 숫자 앞에 삽입하거나 문자, "원"을 숫자 뒤에 삽입할 수 있습니다.

레이아웃은 다음과 같은 형식으로 작성됩니다.

...
<CurrencyFormatter id="cft" precision="1" currencySymbol="ºC" alignSymbol="right"/>
<Column2DChart dataTipFormatter="{cft}">
...

Members

alignSymbolString

currencySymbol 속성을 설정하였을 때, 심볼이 숫자의 어느 위치에 표시될 지를 설정합니다.

유효값은 left, right 입니다.

Default Value:
  • left

currencySymbolString

숫자의 앞 또는 뒤에 추가할 문자(열)를 설정합니다.

일반적으로 통화 단위를 삽입하지만 문자열의 설정도 가능합니다.

Default Value:
  • $

decimalSeparatorToString

소수점을 표현하는 문자를 설정합니다.

기본값(default)은 Dot(.) 입니다.

Default Value:
  • .

idString

식별자(Identifier)를 설정합니다.

식별자(Identifier)는 사용자에 의해 정의된 임의의 고유값입니다.

설정된 식별자(Identifier)를 통해 해당 객체에 접근이 가능합니다.

Default Value:
  • null

precisionNumber

출력할 소수점 이하 최대 자리수를 설정합니다.

값이 -1 이면 표현 가능한 모든 소수점 이하 자리의 수가 출력됩니다.

Default Value:
  • -1

roundingString

반올림 방식을 설정합니다.

유효값은 다음과 같습니다.


  • down - 내림

  • up - 올림

  • nearest - 가까운 값에 대하여 내림/올림

  • none - 원본 데이터 그대로 출력

Default Value:
  • none

thousandsSeparatorToString

천단위 구분을 표현하는 문자를 설정합니다.

기본값(default)은 컴마(,) 입니다.

Default Value:
  • ,

useNegativeSignBoolean

음수를 표현하기 위해서 마이너스 부호(-)를 삽입할 지 여부를 설정합니다.

Default Value:
  • true

useThousandsSeparatorBoolean

천단위 구분자(,)를 삽입할 지 여부를 설정합니다.

Default Value:
  • true
