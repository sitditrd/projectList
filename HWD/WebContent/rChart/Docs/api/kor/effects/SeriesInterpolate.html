Class SeriesInterpolateStyles
 
Packagekor.effects
Classpublic class SeriesInterpolate

SeriesInterpolate 클래스는 축상의 최소값 지점에서 데이터 값에 해당되는 지점까지의 출력에 대한 애니메이션 이펙트를 표현합니다.

<SeriesInterpolate> 의 레이아웃은 다음과 같은 형식으로 작성됩니다.

  <rMateChart>
   ...
    <Line2DSeries ... >
     <showDataEffect>
      <SeriesInterpolate duration="600"/>
     </showDataEffect>
    </Line2DSeries>
   ...
  </rMateChart>


Styles
 StyleDescriptionDefined by
  
duration
Type: Number

이펙트의 총 재생 시간을 밀리 세컨드(Millisecond) 단위로 지정합니다.
모든 이펙트는 duration 속성에 정의된 시간 동안에 재생되고 종료됩니다.

The default value is 500.		SeriesInterpolate
  
elementOffset
Type: Number

한개의 데이터 아이템에 대한 이펙트 재생이 시작된 후 다음 데이터 아이템의 이펙트 재생이 실행되기 까지의 지연시간을 지정합니다.

The default value is 20.		SeriesInterpolate
  
id
Type: String

식별자(Identifier)를 설정합니다. 식별자(Identifier)는 사용자에 의해 정의된 임의의 고유값입니다.
설정된 식별자(Identifier)를 통해 해당 객체에 접근이 가능합니다.

The default value is undefined.		SeriesInterpolate
  
minimumElementDuration
Type: Number

각 데이터 아이템의 이펙트가 완료되는데 필요한 시간을 밀리 세컨드(Millisecond) 단위로 지정합니다.
duration 속성값이 minimumElementDuration 속성값보다 작은 경우 duration 속성은 무시됩니다.

The default value is 0.		SeriesInterpolate
  
offset
Type: Number

시리즈 사이의 이펙트의 지연 시간을 밀리 세컨드(Millisecond) 단위로 지정합니다.
다수의 시리즈를 정의했을 때, 시리즈의 재생 시간에 대한 차이를 주고싶을 경우 이 속성을 사용합니다.

The default value is 0.		SeriesInterpolate
