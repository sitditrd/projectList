Class CheckableLegendItemStyles
 
Packagekor.options
Classpublic class CheckableLegendItem

CheckableLegendItem 클래스는 SubLegend 클래스 인스턴스의 useVisibleCheck 속성값이 true 인 경우 각 범례 항목들을 표현합니다.

레이아웃은 다음과 같은 형식으로 작성됩니다.

  <SubLegend>
   <CheckableLegendItem targetSeries="{series1}" label="Item1"/>
   <CheckableLegendItem targetSeries="{series2}" label="Item2"/>
  </SubLegendItem>
