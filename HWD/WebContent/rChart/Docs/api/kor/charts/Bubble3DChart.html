Class Bubble3DChartStyles
 
Packagekor.charts
Classpublic class Bubble3DChart
InheritanceBubble3DChart Bubble2DChart CartesianChart ChartBase

Bubble3DChart 클래스는 데이터 포인트를 3개의 값(X축 값, Y축 값, 반경)에 따라서 차트상에 버블로 표현합니다.

각 데이터 포인트는 수평 축을 따르는 값, 수직 축을 따르는 값 그리고 버블의 크기 값으로 정의됩니다.
Bubble3DChart 의 series 속성에는 Bubble3DSeries 클래스 인스턴스의 배열을 지정합니다.

레이아웃은 다음과 같은 형식으로 작성됩니다.

  <rMateChart >
   <Options>
    <Caption text="Annual Report"/>
   </Options>
   <Bubble3DChart showDataTips="true" minRadius="5" maxRadius="30">
    <series>
     <Bubble3DSeries displayName="Profit/Cost/Revenue" xField="Profit" yField="Cost" radiusField="Revenue"/>
    </series>
   </Bubble3DChart>
  </rMateChart>

모바일 기기에서 손가락으로 차트 영역을 이동할 경우, touchMove 이벤트를 브라우저에 넘길지 여부를 설정합니다.
touchMove 이벤트를 브라우저에 넘기면 차트 영역에서 touchMove 이벤트 발생 시 브라우저 스크롤이 일어나며,
touchMove 이벤트를 브라우저에 넘기지 않으면 차트 영역 내에서 브라우저 스크롤이 일어나지 않습니다.
이는 차트를 touchMove 할 때 브라우저 스크롤이 발생하면, tooltips 을 확인하기 어려운 문제가 있으므로, 이를 사용자가 선택할 수 있도록 하기 위한 속성입니다.
