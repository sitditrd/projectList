Class Pie3DChartStyles
 
Packagekor.charts
Classpublic class Pie3DChart
InheritancePie3DChart ChartBase

Pie3DChart 클래스는 그라데이션 효과가 있는 3D 형태의 파이 모양으로 데이터를 표현합니다.

삽입된 데이터의 합을 100% 로 계산하고, 각 데이터(슬라이스)의 크기는 전체 데이처(100%)의 비율로 결정됩니다.

Pie3DChart 클래스를 통해서 3D 파이 차트, 3D 도넛 차트, 3D 스택(stacked) 파이 차트를 생성할 수 있습니다.

Pie3DChart 의 series 속성에는 Pie3DSeries 클래스 인스턴스의 배열을 지정합니다.

레이아웃은 다음과 같은 형식으로 작성됩니다.

  <rMateChart >
    <Options>
     <Caption text="Annual Report"/>
   </Options>
   <Pie3DChart showDataTips="true">
     <series>
      <Pie3DSeries field="Profit" nameFiled="Month"/>
     </series>
    </Pie3DChart>
   </rMateChart>

touchMoveEnable
Type: Boolean

모바일 기기에서 손가락으로 차트 영역을 이동할 경우, touchMove 이벤트를 브라우저에 넘길지 여부를 설정합니다.
touchMove 이벤트를 브라우저에 넘기면 차트 영역에서 touchMove 이벤트 발생 시 브라우저 스크롤이 일어나며,
touchMove 이벤트를 브라우저에 넘기지 않으면 차트 영역 내에서 브라우저 스크롤이 일어나지 않습니다.
이는 차트를 touchMove 할 때 브라우저 스크롤이 발생하면, tooltips 을 확인하기 어려운 문제가 있으므로, 이를 사용자가 선택할 수 있도록 하기 위한 속성입니다.
