Class GaugeStyles
 
Packagekor.charts
Classpublic class Gauge

Gauge 클래스는 설정된 최소 각도부터 최대 각도까지의 원형 게이지를 표현합니다. 
  <rMateChart>
   <Options>
    <Caption text="Gauge"/>
   <Options>
   <Gauge innerRatio="0.5" minimumAngle="0" maximumAngle="360" showDataTip="true"
    startAngle="0" showAnimation="true" duration="800" showLabel="true" bounceAnimating="true"/>
  </rMateChart>
