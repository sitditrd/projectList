Package kor.chartsClasses
 

The charts package contains rMate Chart Control classes for the rMate charting components.


Classes
 ClassDescription
 Area2DChart Area2DChart 클래스는 Y축의 시작점부터 데이터 값까지를 영역으로 설정하고 해당 영역에 색상을 칠하는 방식으로 차트를 표현합니다.
 Bar2DChart Bar2DChart 클래스는 데이터의 크기에 맞는 가로 막대(바)로 차트를 표현합니다.
 Bar2DWingChart Bar2DWingChart 클래스는 데이터의 크기에 맞는 가로 막대(바)로 좌우에 하나씩 차트를 표현합니다.
 Bar3DChart Bar3DChart 클래스는 데이터의 값에 따라 일련의 큐브 형태의 가로 막대(바)로 차트를 표현합니다.
 BoxPlotChart BoxPlotChart 클래스는 주어진 데이터에서 각 사분위수를 계산하여 통계 요약수치를 차트로 표현합니다.
 Bubble2DChart Bubble2DChart 클래스는 데이터 포인트를 3개의 값(X축 값, Y축 값, 반경)에 따라서 차트상에 버블로 표현합니다.
 Bubble3DChart Bubble3DChart 클래스는 데이터 포인트를 3개의 값(X축 값, Y축 값, 반경)에 따라서 차트상에 버블로 표현합니다.
 CandleArea2DChart CandleArea2DChart 클래스는 공시 데이터를 표현할 수 있는 CandleArea2DSeries 를 출력하는 차트 입니다.
 CandleLine2DChart CandleLine2DChart 클래스는 공시 데이터를 표현할 수 있는 CandleLine2DSeries 를 출력하는 차트 입니다.
 Candlestick2DChart Candlestick2DChart 클래스는 시가, 종가, 고가, 저가에 대한 데이터를 차트에 표현합니다.
 CircularGauge CircularGauge 클래스는 360 도 원형 게이지를 표현합니다.
 Column2DChart Column2DChart 클래스는 데이터의 크기에 맞는 세로 막대(칼럼)로 차트를 표현합니다.
 Column2DWingChart Column2DWingChart 클래스는 데이터의 크기에 맞는 세로 막대(칼럼)로 상하에 하나씩 차트를 표현합니다.
 Column3DChart Column3DChart 클래스는 데이터의 값에 따라 일련의 큐브 형태의 세로 막대(칼럼)로 차트를 표현합니다.
 Combination2DChart Combination2DChart 클래스는 여러가지 유형의 2D 차트를 조합해 놓은 형태로 데이터를 표현할 때 유용한 차트입니다.
 Combination3DChart Combination3DChart 클래스는 여러가지 유형의 2D, 3D 차트를 조합해 놓은 형태로 데이터를 표현할 때 유용한 차트입니다.
 Displayer Displayer 클래스는 히스토리 차트에서 선택 영역을 보여주는 2D 디스플레이어 차트를 표현합니다.
 Displayer3D The Displayer3D class represents a 3D display chart that shows the selected dataset in the history chart.
 DualChart DualChart 클래스는 mainChart, subChart, dataSelector 로 구성됩니다.
 Equalizer2DChart Equalizer2DChart 클래스는 데이터의 크기에 맞는 이퀄라이저 형태의 세로 막대(칼럼)로 차트를 표현합니다.
 Gauge Gauge 클래스는 설정된 최소 각도부터 최대 각도까지의 원형 게이지를 표현합니다.
 HalfCircularGauge HalfCircularGauge 클래스는 180 도에서 360 도까지 구간의 반원형 게이지를 표현합니다.
 HCylinderGauge HCylinderGauge 클래스는 가로 방향의 실린더 모양으로 목표값과 실적값을 표현합니다.
 Histogram2DChart Histogram2DChart 클래스는 차트에 입력되는 데이터를 특정 구간별로 그룹화하여 2D 형태의 막대로 차트를 표현합니다.
 Histogram3DChart Histogram3DChart 클래스는 차트에 입력되는 데이터를 특정 구간별로 그룹화하여 3D 형태의 막대로 차트를 표현합니다.
 HistogramCategoryAxis HistogramCategoryAxis 클래스는 히스토그램 차트(Histogram2DChart)의 축 생성에 적용됩니다.
 HistoryChart HistoryChart 클래스는 디스플레이어(Displayer), 네비게이터(Navigator), 셀렉터(Selector)로 구성되어 있습니다.
 HistoryRangeSelector HistoryRangeSelector 클래스는 히스토리 차트의 셀렉터를 표현합니다.
 HLinearGauge HLinearGauge 는 가로 방향의 막대 모양과 목표값을 나타내는 선으로 구성되어 있습니다.
 ImageChart ImageChart 클래스는 데이터의 값을 이미지의 크기로 차트를 표현합니다.
 ImageMatrixChart ImageMatrixChart 클래스는 데이터의 개수에 따라서 설정한 이미지를 반복적으로 출력하여 차트를 표현합니다.
 Line2DChart Line2DChart 클래스는 각 데이터 포인트를 선으로 연결하여 차트를 표현합니다.
 LinearGauge LinearGauge 클래스는 설정된 막대의 방향과 값에 따라서 막대(가로, 세로) 게이지를 표현합니다.
 Matrix2DChart Matrix2DChart 클래스는 데이터가 표현되는 위치를 X축 값, Y축 값으로 설정하고, 데이터의 크기는 Z축 값에 의해서 차트를 표현합니다.
 MotionChart MotionChart 클래스는 슬라이더 기능을 이용한 애니메이션 효과를 통하여 데이터의 변동을 차트에 표현합니다.
 Navigator Navigator 클래스는 히스토리 차트에서 전체 데이터를 표시하는 네비게이터 차트를 표현합니다.
 OverlayBubbleChart OverlayBubbleChart 클래스는 데이터를 겹쳐진 버블 형태의 차트로 표현합니다.
 Pie2DChart Pie2DChart 클래스는 그라데이션 효과가 있는 2D 형태의 파이 모양으로 데이터를 표현합니다.
 Pie3DChart Pie3DChart 클래스는 그라데이션 효과가 있는 3D 형태의 파이 모양으로 데이터를 표현합니다.
 Plot2DChart Plot2DChart 클래스는 X축과 Y축 구간의 데이터 포인트를 심볼로 표현합니다.
 Pyramid2DChart Pyramid2DChart 클래스는 차트에 설정된 데이터를 다각형 형태로 표현합니다.
 RadarChart RadarChart 클래스는 카테고리 수에 따라 원을 같은 간격으로 나누고 각 카테고리 별 값에 따라 해당 위치에 점을 찍고 점을 이은 선으로 차트를 표현합니다.
 RealTimeChart RealTimeChart 클래스는 수신되는 데이터를 시간 혹은 수량 기준으로 차트로 표현합니다.
 TreeMapChart TreeMapChart 클래스는 Tree 형태의 XML 데이터를 각 데이터의 크기에 따라 사각형 모양으로 표시하여 차트를 표현합니다.
 VCylinderGauge VCylinderGauge 는 클래스는 세로 방향의 실린더에 목표값과 실적값을 표현합니다.
 Vector2DChart Vector2DChart 는 데이터에 대한 각도 값과 크기 값을 받아서 각 데이터를 선으로 표현합니다.
 VLinearGauge VLinearGauge 는 클래스는 세로 방향의 막대에 목표값과 실적값을 표현합니다.
 WindRoseChart WindRoseChart 클래스는 일반적으로 특정 위치에서 수집된 바람의 속도와 방향 데이터를 차트로 표현합니다.
 WordCloudChart WordCloudChart 클래스는 데이터로 지정된 문자열의 반복 횟수 혹은 문자열에 지정된 가중치(데이터 크기)에 따라서 차트를 표현합니다.
