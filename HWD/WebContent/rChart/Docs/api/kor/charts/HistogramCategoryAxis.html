Class HistogramCategoryAxisStyles
 
Packagekor.charts
Classpublic class HistogramCategoryAxis

HistogramCategoryAxis 클래스는 히스토그램 차트(Histogram2DChart)의 축 생성에 적용됩니다. 
  <rMateChart >
    <Options>
     <Caption text="Annual Report"/>
   </Options>
   <Histogram2DChart showDataTips="true">
     <horizontalAxis>
      <HistogramCategoryAxis categoryField="histogramXField" />
     </horizontalAxis>
     <series>
      <Histogram2DSeries yField="Profit"/>
     </series>
    </Histogram2DChart>
   </rMateChart>


Styles
 StyleDescriptionDefined by
  
categoryField
Type: String

카테고리 축의 라벨로 표시될 데이터 필드명을 지정합니다.
예를 들어, 수평축(horizontalAxis)에 HistogramCategoryAxis를 설정하였다면 X축 라벨은 categoryField 속성에 지정한 데이터가 출력됩니다.

The default value is "histogramXField".		HistogramCategoryAxis
  
displayName
Type: String

축의 이름을 지정합니다. 설정된 값은 기본적으로 데이터팁(툴팁)에서 보여지게 됩니다.

HistogramCategoryAxis
  
id
Type: String

식별자(Identifier)를 설정합니다. 식별자(Identifier)는 사용자에 의해 정의된 임의의 고유값입니다.
설정된 식별자(Identifier)를 통해 해당 객체에 접근이 가능합니다.

HistogramCategoryAxis
  
labelJsFunction
Type: String

축 라벨을 사용자 정의 라벨로 표현하기 위한 자바스크립트 함수명을 지정합니다.
차트 레이아웃 설정시 labelJsFunction 속성값을 자바스크립트 함수명으로 지정합니다.

  예) <horizontalAxis>
         <HistogramCategoryAxis id="hAxis" categoryField="Month" labelJsFunction="axisLabelFunc" />
     </horizontalAxis>

labelJsFunction 함수 시그너쳐(signature)는 다음과 같습니다. 

  function_name(id, value) { ... }
    id : 축의 식별자(Identifier).
  value : 축의 라벨값.
HistogramCategoryAxis
  
padding
Type: Number

차트의 시리즈 아이템이 출력되는 곳의 여백 크기를 설정합니다.
칼럼, 바 차트 계열의 기본(default) padding 값은 0.5 입니다.
라인, 영역 차트의 padding 값을 0으로 설정한 경우, 차트의 시리즈 아이템이 출력되는 곳의 여백은 없습니다.
여백의 단위는 차트의 시리즈 아이템 하나의 공간이 1 입니다.

The default value is 0.5.		HistogramCategoryAxis
  
ticksBetweenLabels
Type: Boolean

축 라벨을 눈금과 눈금 사이에 표시할 지 눈금과 같은 위치에 표시할 지 여부를 나타냅니다.
true 인 경우 눈금과 눈금 사이에 라벨을 표시합니다.

The default value is false.		HistogramCategoryAxis
  
title
Type: String

축의 제목을 지정합니다. 세로축인 경우 세로로 제목이 출력됩니다.
