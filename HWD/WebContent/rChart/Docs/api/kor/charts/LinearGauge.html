Class LinearGaugeStyles
 
Packagekor.charts
Classpublic class LinearGauge

LinearGauge 클래스는 설정된 막대의 방향과 값에 따라서 막대(가로, 세로) 게이지를 표현합니다. 
  <rMateChart>
   <Options>
    <Caption text="Gauge"/>
   <Options>
   <LinearGauge minimum="0" maximum="100" showDataTip="true" direction="vertical"
    showAnimation="true" duration="800" showLabel="true" bounceAnimating="true"/>
  </rMateChart>
