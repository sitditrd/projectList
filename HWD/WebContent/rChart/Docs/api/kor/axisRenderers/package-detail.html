Package kor.axisRenderersClasses
 

The axisRenderers package contains the classes for the rMate Chart Axis 2D, 3D Renderer classes


Classes
 ClassDescription
 AngularAxisRenderer AngularAxisRenderer 클래스는 레이더 차트에서 카테고리에 해당되는 축을 렌더링합니다.
 Axis2DRenderer Axis2DRenderer 클래스는 차트의 종류가 2D인 경우, 축의 렌더링에 사용됩니다.
 Axis2DWingRenderer 차트의 종류가 Column2DWingChart 혹은 Bar2DWingChart 인 경우, 축의 렌더링 방식을 설정합니다.
 Axis3DRenderer Axis3DRenderer 클래스는 차트의 종류가 3D인 경우, 축의 렌더링에 사용됩니다.
 BrokenAxis2DRenderer BrokenAxis2DRenderer 클래스는 차트의 종류가 2D이고 BrokenAxis를 사용할 경우 축의 렌더링에 사용됩니다.
 BrokenAxis3DRenderer BrokenAxis3DRenderer 클래스는 차트의 종류가 3D이고 BrokenAxis를 사용할 경우 축의 렌더링에 사용됩니다.
 HistogramAxis2DRenderer HistogramAxis2DRenderer 클래스는 Histogram2DChart를 생성할 경우 축의 렌더링에 사용됩니다.
 ScrollableAxisRenderer ScrollableAxisRenderer 클래스는 차트의 종류가 스크롤이 가능한 차트일 경우 축의 렌더링에 사용됩니다.
