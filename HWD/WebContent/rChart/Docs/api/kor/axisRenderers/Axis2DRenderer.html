Class Axis2DRendererStyles
 
Packagekor.axisRenderers
Classpublic class Axis2DRenderer
SubclassesAxis2DWingRenderer, Axis3DRenderer, BrokenAxis2DRenderer, HistogramAxis2DRenderer, ScrollableAxisRenderer

Axis2DRenderer 클래스는 차트의 종류가 2D인 경우, 축의 렌더링에 사용됩니다.

눈금(tick) - 축에서 데이터들을 구분하기 위해 표시하는 선.
축의 굵기, 축의 색, 눈금의 위치, 눈금의 색 등을 조절할 수 있습니다.
반드시 axis 속성에 해당 축을 지정해야 합니다.

레이아웃은 다음과 같은 형식으로 작성됩니다.

  <Axis2DRenderer axis="{vAxis}" tickLength="20" minorTickLength="0" tickPlacement="outside" placement="left"
         canDropLabels="false" showLabels="true">
      <axisStroke>
       <Stroke weight="10" color="0x66CCFF" />
      </axisStroke>
      <tickStroke>
       <Stroke weight="2" color="0xFF0000" />
     </tickStroke>
      <minorTickStroke>
       <Stroke weight="1" color="0x0000FF" caps="square"/>
     </minorTickStroke>
   </Axis2DRenderer>

canDropLabels
Type: Boolean

축의 라벨들을 겹치지 않게 표시하기 위해 필요에 따라서 라벨들을 삭제할 지 여부를 설정합니다.
true로 설정하면, Axis2DRenderer 클래스 인스턴스는 렌더링시 겹치는 라벨들을 삭제합니다.
라벨들은 항상 일정한 간격으로 표시되며 필요한 경우 2개 또는 3개의 라벨을 표시하지 않고 다음 라벨을 표시합니다.
연속으로 삭제되는 라벨의 개수는 동일합니다. 예를 들어, 2개의 라벨이 삭제되었다가 다음에는 3개의 라벨이 삭제되지는 않습니다.
false로 설정하고 라벨이 겹치는 경우, Axis2DRenderer 클래스 인스턴스는 회전, 확대/축소를 사용해 라벨을 표시합니다. (내장 폰트 사용시)
이 속성값이 명시적으로 설정되지 않으면, 렌더링하는 축의 유형에 따라서 적절한 값이 기본값(default)으로 설정됩니다.

The default value is true.		Axis2DRenderer

labelJsFunction
Type: String

축 라벨을 사용자 정의 라벨로 표현하기 위한 자바스크립트 함수명을 지정합니다.

  function_name(id, value) { ... }

  value : 축의 라벨값.

minorTickPlacement
Type: String

보조 눈금(minor tick)을 표시하는 위치를 지정합니다.

유효값은 아래와 같습니다.
  • "inside" - 보조 눈금을 데이터 영역 내에 표시합니다.
  • "outside" - 보조 눈금을 라벨 영역 내에 표시합니다.
  • "cross" - 보조 눈금을 축 상에 표시합니다.
  • "none" - 보조 눈금을 표시하지 않습니다.
The default value is none.		Axis2DRenderer

Axis2DRenderer
  
placement
Type: String

차트에서 축이 표시될 위치를 설정하는 속성입니다.
유효한 값은 수직 축인 경우, "left" 와 "right", 수평 축인 경우, "top" 와 "bottom" 입니다.
기본값(default)은 첫번째 축이 왼쪽(수직 축)과 아래(수평 축)이며, 두번째 축이 오른쪽과 위에 배치됩니다.
CartesianChart 에서는 자동적으로 두번째 축이 첫번째 축의 반대 방향에 위치하게 됩니다.
예를 들어, 첫번째 수직 축을 명시적으로 오른쪽에 배치하면, 두번째 수직 축은 왼쪽에 배치됩니다.
