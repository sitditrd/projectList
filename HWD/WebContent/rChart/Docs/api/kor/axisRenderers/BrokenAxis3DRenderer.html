Packagekor.axisRenderers
Classpublic class BrokenAxis3DRenderer
InheritanceBrokenAxis3DRenderer BrokenAxis2DRenderer Axis2DRenderer

BrokenAxis3DRenderer 클래스는 차트의 종류가 3D이고 BrokenAxis를 사용할 경우 축의 렌더링에 사용됩니다.

눈금(tick) - 축에서 데이터들을 구분하기 위해 표시하는 선.
축의 굵기, 축의 색, 눈금의 위치, 눈금의 색 등을 조절할 수 있습니다.
반드시 axis 속성에 해당 축을 지정해야 합니다.

레이아웃은 다음과 같은 형식으로 작성됩니다.

  <BrokenAxis3DRenderer axis="{vAxis}" tickLength="20" minorTickLength="0" tickPlacement="outside" placement="left"
         canDropLabels="false" showLabels="true" labelAlign="center">
   <axisStroke>
    <Stroke weight="10" color="0x66CCFF" caps="none"/>
   </axisStroke>
    <tickStroke>
   <Stroke weight="2" color="0xFF0000" />
   </tickStroke>
   <minorTickStroke>
    <Stroke weight="1" color="0x0000FF" caps="square"/>
   </minorTickStroke>
  </BrokenAxis3DRenderer>

canDropLabels
Type: Boolean

축의 라벨들을 겹치지 않게 표시하기 위해 필요에 따라서 라벨들을 삭제할 지 여부를 설정합니다.
true로 설정하면, Axis2DRenderer 클래스 인스턴스는 렌더링시 겹치는 라벨들을 삭제합니다.
라벨들은 항상 일정한 간격으로 표시되며 필요한 경우 2개 또는 3개의 라벨을 표시하지 않고 다음 라벨을 표시합니다.
연속으로 삭제되는 라벨의 개수는 동일합니다.

기본값(default)은 true 이며 위, 아래 교대로 표시합니다.
