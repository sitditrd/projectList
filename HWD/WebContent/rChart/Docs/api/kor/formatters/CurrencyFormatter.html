Class CurrencyFormatterStyles
 
Packagekor.formatters
Classpublic class CurrencyFormatter

CurrencyFormatter 클래스는 일반적으로 통화 금액을 표현하는 목적으로 활용됩니다.

숫자 표현 형식에 대한 변경 및 숫사 앞, 뒤에 문자(열)의 삽입을 지원합니다.
예를 들어, 통화 단위 "$" 를 숫자 앞에 삽입하거나 문자, "원"을 숫자 뒤에 삽입할 수 있습니다.

currencySymbol 속성을 설정하였을 때, 심볼이 숫자의 어느 위치에 표시될 지를 지정합니다.
유효값은 "left", "right" 입니다.

currencySymbol
Type: String

숫자의 앞 또는 뒤에 추가할 문자(열)를 지정합니다.
일반적으로 통화 단위를 삽입하지만 문자열의 지정도 가능합니다.

The default value is "$".		CurrencyFormatter
  
decimalSeparatorTo
Type: String

소수점을 표현하는 문자를 지정합니다.
기본값(default)은 Dot(.) 입니다.

The default value is ".".		CurrencyFormatter
  
id
Type: String

식별자(Identifier)를 설정합니다. 식별자(Identifier)는 사용자에 의해 정의된 임의의 고유값입니다.
설정된 식별자(Identifier)를 통해 해당 객체에 접근이 가능합니다.

CurrencyFormatter
  
precision
Type: Number

출력할 소수점 이하 최대 자리수를 지정합니다.
값이 -1 이면 표현 가능한 모든 소수점 이하 자리의 수가 출력됩니다.

The default value is -1.		CurrencyFormatter
  
rounding
Type: String

반올림 방식을 설정합니다.

유효값은 "down", "nearest", "up", "none" 입니다.
  • down - 내림
  • up - 올림
  • nearest - 가까운 값에 대하여 내림/올림
  • none - 원본 데이터 그대로 출력
The default value is "none".		CurrencyFormatter
