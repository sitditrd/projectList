Class NumberFormatterStyles
 
Packagekor.formatters
Classpublic class NumberFormatter

NumberFormatter 클래스는 원하는 숫자의 형식으로 출력하는 것을 지원합니다.

예를 들어, 천단위 구분자를 삽입하거나, 소수점 2자리에서 반올림하는 등의 처리를 할 수 있습니다.


Styles
 StyleDescriptionDefined by
  
decimalSeparatorTo
Type: String

소수점을 표현하는 문자를 지정합니다. 기본값(default)은 Dot(.) 입니다.

The default value is ".".		NumberFormatter
  
id
Type: String

식별자(Identifier)를 설정합니다. 식별자(Identifier)는 사용자에 의해 정의된 임의의 고유값입니다.
설정된 식별자(Identifier)를 통해 해당 객체에 접근이 가능합니다.

NumberFormatter
  
precision
Type: Number

출력할 소수점 이하 최대 자리수를 지정합니다. 값이 -1 이면 표현 가능한 모든 소수점 이하 자리의 수가 출력됩니다.

The default value is -1.		NumberFormatter
  
rounding
Type: String

반올림 방식을 설정합니다.

유효값은 "down", "nearest", "up", "none" 입니다.
  • down - 내림
  • up - 올림
  • nearest - 가까운 값에 대하여 내림/올림
  • none - 원본 데이터 그대로 출력
The default value is "none".		NumberFormatter
  
thousandsSeparatorTo
Type: String

천단위 구분을 표현하는 문자를 지정합니다. 기본값(default)은 컴마(,) 입니다.

The default value is ",".		NumberFormatter
  
useNegativeSign
Type: Boolean

음수를 표현하기 위해서 마이너스 부호(-)를 삽입할 지 여부를 설정합니다.

The default value is true.		NumberFormatter
  
useThousandsSeparator
Type: Boolean

천단위 구분자(,)를 삽입할 지 여부를 설정합니다.

The default value is true.		NumberFormatter
