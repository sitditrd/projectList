Class DateFormatterStyles
 
Packagekor.formatters
Classpublic class DateFormatter

DateFormatter 클래스는 원하는 날짜의 형식으로 출력하는 것을 지원합니다.

차트에서 처리되는 기본 날짜 형식은 YYYY/MM/DD HH:NN:SS 입니다.
차트에서 처리되는 기본 날짜 형식이 아니면 일반 스트링으로 처리되기 때문에, DateFormatter 클래스를 적용하기 위해서는 데이터 형식이 기본 날짜 형식과 일치하도록 해야합니다.

formatString

출력하고자 하는 날짜 형식을 지정합니다.

  ┌──────────────────────────────────┐
  │ Y │ 연(Year)  │ YY=05,YYYY=2005  │
  │ M │ 월(Month) │ M=7,MM=07,MMM=Jul│
  │ D │ 일(Day)   │ D=4,DD=05        │
  │ H │ 시(Hour)  │ H=1,HH=01        │
  │ N │ 분(Minute)│ N=1,NN=01        │
  │ S │ 초(Second)│ SS=30            │
  └──────────────────────────────────┘
예를 들어, 날짜 형식을 "2009년 3월" 과 같이 표현하고자 하는 경우 formatStirng="YYYY년 M월" 이 됩니다. 		DateFormatter
