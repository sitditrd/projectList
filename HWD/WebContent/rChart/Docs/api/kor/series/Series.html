Class SeriesStyles
 
Packagekor.series
Classpublic class Series
SubclassesArea2DSet, Bar2DSet, CartesianSeries, Column2DSet, ImageMatrixSeries, ImageMatrixSet, OverlayBubbleSeries, Pie2DSeries, Pyramid2DSeries, RadarSeries, TreeMapSeries, WindRoseSeries, WindRoseSet, WingSeries

Series 클래스는 모든 시리즈 클래스들의 기본 클래스입니다.

모든 차트들의 시리즈 클래스들은 Series 클래스를 상속받기 때문에 사용자는 이를 레이아웃에 설정하지 않아도 됩니다.


Styles
 StyleDescriptionDefined by
  
displayName
Type: String

시리즈의 이름을 지정합니다.
설정된 값은 기본적으로 데이터팁(툴팁)에서 보여지게 되고 범례의 항목으로 표시됩니다.

Series
  
fill
Type: Object

시리즈 아이템렌더러의 채우기 색을 지정합니다.

Series
  
fillJsFunction
Type: String

사용자 정의 색상을 설정하는 자바스크립트 함수명을 지정합니다.
차트 레이아웃 설정시 Series 클래스 인스턴스의 fillJsFunction 속성값에 자바스크립트 함수명을 지정합니다.

  예) <Column2DSeries yField="Profit" fillJsFunction="fillJsFunc">

fillJsFunction 함수 시그너쳐(signature)는 다음과 같습니다.

  function_name(seriesId, index, data, values) { ... }
seriesId : 시리즈 식별자(Identifier)
index : 데이터 아이템의 인덱스
data : 데이터 아이템의 데이터 객체
values : 데이터 아이템의 데이터 배열. 차트의 종류에 따라 아래와 같습니다.
  • 바 시리즈(Bar2DSeries, Bar3DSeries) 0 : x축 값, 1 : y축 값
  • 칼럼 시리즈(Column2DSeries, Column3DSeries) 0 : x축 값, 1 : y축 값
  • 영역 시리즈(Area2DSeries) 0 : x축 값, 1 : y축 값
  • 버블 시리즈(Bubble3DSeries) 0 : x축 값, 1 : y축 값, 2 : radius 값
  • 라인 시리즈(Line2DSeries) 0 : x축 값, 1 : y축 값
  • 파이 시리즈(Pie2DSeries, Pie3DSeries) 0 : 값, 1 : 퍼센티지, 2 : nameField
  • From-To Chart 에서 minField 를 지정했다면 values 의 마지막 인덱스 값에 minField 값이 설정됩니다.
  • 매트릭스 시리즈 (Matrix2DSeries) 0 : x축 값, 1 : y축 값, 2 : z축 값
  • 캔들 시리즈 (Candlestick2DSeries) 0 : x축 값, 1 : openField 값, 2 : closeField 값, 3 : highField 값, 4 : lowField 값
  • 윙 시리즈(Column2DWingSeries, Bar2DWingSeries) 0 : x축 값, 1 : y축 값, 2 : xOpp 값, 3 : yOpp 값
  • 히스토그램 시리즈(Histogram2D, Histogram3DSeries) 0 : x축 값, 1 : y축 값
  • 트리맵 시리즈(TreeMapSeries) 0 : weight 값, 1 : text 값
  • 박스플롯 시리즈(BoxPlotSeries) 0 : upper 값, 1 : median 값, 2 : lower 값, 3 : min 값, 4 : hidden 값
  • 벡터 시리즈(Vector2DSeries) 0 : degree 값, 1 : velocity 값, 2 : meter 값
  • 워드 클라우드 시리즈(WordCloudSeries) 0 : text 값, 1 : weight 값
  • 이퀄라이저 시리즈(Equalizer2DSeries) 0 : x축 값, 1 : y축 값
  • 오버레이 버블 시리즈(OverlayBubbleSeries) 0 : field 값, 1 : nameField 값
  • 피라미드 시리즈(Pyramid2DSeries) 0 : weight 값, 1 : nameField 값
  • 윈드 로즈 시리즈(WindRoseSeries) 0 : field 값
Series
  
fills
Type: Array

시리즈에서 각각의 아이템렌더러의 fill 컬러를 지정하는 fill 객체 배열을 정의합니다.
만약 시리즈의 모든 아이템렌더러에 충분한 fill 컬러를 지정하지 않았다면 지정된 색깔이 반복됩니다.
예를 들어, 시리즈의 아이템이 10개 인 경우 5개의 fill 컬러를 지정하였다면 5개를 칠한 후 다시 5개를 칠하게 됩니다.

     <...2DSeries ... >
      <fills>
       <SolidColor color="0xCC66FF"/>
       <SolidColor color="0x9966CC"/>
       <SolidColor color="0x9999CC"/>
      </fills>
     </...2DSeries>

설정된 식별자(Identifier)를 통해 해당 객체에 접근이 가능합니다.
