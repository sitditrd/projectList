Class CandleArea2DSeriesStyles
 
Packagekor.series
Classpublic class CandleArea2DSeries
InheritanceCandleArea2DSeries Area2DSeries CartesianSeries Series

CandleArea2DSeries 클래스는 CandleArea2DChart 클래스 인스턴스의 series 속성에 설정됩니다.

차트에 설정된 데이터를 처리하고 캔들 영역 차트를 표현합니다.

레이아웃은 다음과 같은 형식으로 작성됩니다.

  <rMateChart >
    <Options>
     <Caption text="Annual Report"/>
   </Options>
   <CandleArea2DChart showDataTips="true">
     <horizontalAxis>
      <CategoryAxis categoryField="Month" />
     </horizontalAxis>
     <series>
      <CandleArea2DSeries yField="Profit"/>
      <CandleArea2DSeries yField="Cost" />
      <CandleArea2DSeries yField="Revenue" />
     </series>
    </CandleArea2DChart>
   </rMateChart>
