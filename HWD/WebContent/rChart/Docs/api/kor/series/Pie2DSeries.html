Pie2DSeries 클래스는 Pie2DChart 클래스 인스턴스의 series 속성에 설정됩니다.

차트에 설정된 데이터를 처리하고 파이를 표현합니다.

레이아웃은 다음과 같은 형식으로 작성됩니다.

  <rMateChart >
    <Options>
     <Caption text="Annual Report"/>
   </Options>
   <Pie2DChart showDataTips="true">
     <series>
      <Pie2DSeries field="Profit" nameField="Month"/>
     </series>
    </Pie2DChart>
   </rMateChart>

fills
Type: Object

시리즈의 아이템 렌더러가 아이템에 채울 색상 배열(SolidColor 클래스 인스턴스 배열)을 설정합니다.
만약 시리즈의 모든 아이템에 각각의 SolidColor 클래스 인스턴스가 설정되지 않으면, 설정된 클래스 인스턴스들이 반복 적용됩니다.
예를 들어, 시리즈의 아이템이 10개이고 5개의 SolidColor 클래스 인스턴스가 설정되었으면, 6번째부터 10번째 시리즈 아이템에는 정의된 5개의 SolidColor 클래스 인스턴스가 한번 더 적용됩니다.

     <Plot2DSeries ... >
      <fills>
       <SolidColor color="0xCC66FF"/>
       <SolidColor color="0x9966CC"/>
       <SolidColor color="0x9999CC"/>
      </fills>
     </Plot2DSeries>

labelJsFunction
Type: String

파이 차트에서 labelPosition 속성을 설정한 경우, 사용자 정의 라벨(수치값)을 설정하는 자바스크립트 함수명을 지정합니다.
차트 레이아웃 설정시 Series 클래스 인스턴스의 labelJsFunction 속성값에 자바스크립트 함수명을 지정합니다.

  예) <Pie2DSeries field="Profit" nameField="Month" labelPosition="inside" labelJsFunction="pieSeriesLabelFunc" />

labelJsFunction 함수 시그너쳐(signature)는 다음과 같습니다.

  function_name(seriesId, index, data, values) { ... }
seriesId : 레이아웃에서 부여한 시리즈의 식별자(Identifier)가 있을 경우 해당 식별자를 나타냅니다.
index : 해당 아이템(막대, 파이 조각 등)의 인덱스 번호 - 가장 왼쪽 또는 가장 위쪽의 인덱스 번호는 0 입니다.
data : 해당 아니템의 값을 표현하기 위해 입력된 데이터. (데이터의 종류에 따라서 XML의 내용 또는 배열이 됩니다.)
values : 해당 아이템의 값입니다. 배열로 전달되며 차트의 종류에 따라 아래와 같습니다.
  • 파이시리즈(Pie2DSeries, Pie3DSeries) 0 : 값, 1:퍼센티지, 2 : nameField
Pie2DSeries
  
labelPosition
Type: String

라벨(수치값)을 표시하는 방법을 지정합니다.

유효값은 다음과 같습니다.
  • "none" - 라벨을 표시하지 않습니다.
  • "outside" - 차트의 가장자리에 라벨을 표시합니다.
  • "callout" - 차트 외부에 라벨을 표시하고 해당되는 슬라이스를 선으로 연결합니다.
  • "inside" - 차트 내부의 해당 슬라이스에 라벨을 표시합니다.
  • "insideWithCallout" - 기본적으로 차트 내부의 슬라이스에 라벨을 표시하고, 만약 라벨이 겹치면 해당 슬라이스의 라벨은 callout 방법으로 표시합니다.
The default value is "none".		Pie2DSeries
