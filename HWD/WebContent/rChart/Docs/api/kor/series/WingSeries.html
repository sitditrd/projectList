Class WingSeriesStyles
 
Packagekor.series
Classpublic class WingSeries
InheritanceWingSeries Series
SubclassesBar2DWingSeries, Column2DWingSeries

WingSeries 클래스는 Bar2DWingSeries, Column2DWingSeries 클래스들의 기본 클래스입니다.

윙 시리즈 클래스들이 WingSeries 클래스를 상속받기 때문에 사용자는 이를 레이아웃에 설정하지 않아도 됩니다.
