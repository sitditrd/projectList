Class CartesianSeriesStyles
 
Packagekor.series
Classpublic class CartesianSeries
InheritanceCartesianSeries Series
SubclassesArea2DSeries, Bar2DSeries, Bubble2DSeries, Column2DSeries, Line2DSeries, Matrix2DSeries, Plot2DSeries, Vector2DSeries, WordCloudSeries

CartesianSeries 클래스는 카테시안 계열(Line2DSeries, Column2DSeries 등) 시리즈들의 기본 클래스입니다.

카테시안 계열 시리즈 클래스들이 CartesianSeries 클래스를 상속받기 때문에 사용자는 이를 레이아웃에 설정하지 않아도 됩니다.

sortOnXField
Type: Boolean

차트에서 사용될 데이터에 대해서 X축 데이터를 기준으로 자동으로 정렬할 지 여부를 설정합니다.
값이 false 이면 데이터가 입력된 순서로 차트에 표현됩니다.

1. X축은 숫자 축(LinearAxis)이어야 합니다.
2. 시리즈의 xField 속성(X축 데이터)에 데이터 필드명을 지정합니다.
3. 예) sortOnXField 설정에 따라서 차트에 표현되는 데이터의 순서가 달라집니다.
4. {"data":4},{"data":1},{"data":2},{"data":3} 순으로 데이터가 입력되었을 경우
- sortOnXField 값이 true 이면, 차트에는 1, 2, 3, 4 의 순서로 데이터가 표현되고,
- sortOnXField 값이 false 이면, 차트에는 4, 1, 2, 3 의 순서로 데이터가 표현됩니다.

The default value is true.		CartesianSeries
