Class Column3DSeriesStyles
 
Packagekor.series
Classpublic class Column3DSeries
InheritanceColumn3DSeries Column2DSeries CartesianSeries Series
SubclassesHistogram3DSeries, HTarget3DResultSeries, VTarget3DGoalSeries, VTarget3DResultSeries

Column3DSeries 클래스는 Column3DChart 클래스 인스턴스의 series 속성에 설정됩니다.

차트에 설정된 데이터를 처리하고 3D 세로 막대(칼럼, Column) 차트를 표현합니다.

레이아웃은 다음과 같은 형식으로 작성됩니다.

  <rMateChart >
    <Options>
     <Caption text="Annual Report"/>
   </Options>
   <Column3DChart showDataTips="true">
     <horizontalAxis>
      <CategoryAxis categoryField="Month" />
     </horizontalAxis>
     <series>
      <Column3DSeries yField="Profit"/>
      <Column3DSeries yField="Cost" />
      <Column3DSeries yField="Revenue" />
     </series>
    </Column3DChart>
   </rMateChart>
