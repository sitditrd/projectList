Classpublic class TreeMapSeries
InheritanceTreeMapSeries Series

TreeMapSeries 클래스는 TreeMapChart 클래스 인스턴스의 series 속성에 설정됩니다.

차트에 설정된 데이터를 처리하고 트리맵 차트를 표현합니다.

레이아웃은 다음과 같은 형식으로 작성됩니다.

  <rMateChart >
   <Options>
    <Caption text="Annual Report"/>
  </Options>
  <TreeMapChart showDataTips="true">
    <series>
     <TreeMapSeries weightField="weight" nameField="name" groupField="group" groupNameField="groupName"/>
    </series>
   </TreeMapChart>
  </rMateChart>
