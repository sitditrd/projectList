Classpublic class Pyramid2DSeres
InheritancePyramid2DSeres Series

Pyramid2DSeres 클래스는 Pyramid2DChart 클래스 인스턴스의 series 속성에 설정됩니다.

차트에 설정된 데이터를 처리하고 피라미드 차트를 표현합니다.

레이아웃은 다음과 같은 형식으로 작성됩니다.

     <rMateChart >
      <Pyramid2DChart ... >
          <series>
              <Pyramid2DSeries weightField="data" nameField="name" leftTopRatio="0" rightTopRatio="0" 
                  leftBottomRatio="1" rightBottomRatio="1"/>
          </series>
      </Pyramid2DChart>
     </rMateChart>
