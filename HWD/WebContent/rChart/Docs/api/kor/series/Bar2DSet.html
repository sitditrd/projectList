Class Bar2DSetStyles
 
Packagekor.series
Classpublic class Bar2DSet
InheritanceBar2DSet Series
SubclassesBar3DSet

Bar2DSet 클래스는 다중의 Bar2DSeries 클래스 인스턴스를 하나의 대표 시리즈로 표현합니다.

Bar2DSet 으로 대표되는 복수의 Bar2DSeries 는 마치 하나의 시리즈와 같이 작동됩니다.

레이아웃은 다음과 같은 형식으로 작성됩니다.

  <Combination2DChart showDataTips="true">
   <verticalAxis>
    <CategoryAxis categoryField="Month"/>
   </verticalAxis>
   <series>
       <Bar2DSet type="clustered">
        <series>
      <Bar2DSeries xField="Cost" displayName="Cost"/>
      <Bar2DSeries xField="Revenue" displayName="Revenue"/>
     </series>
    </Bar2DSet>
    <Line2DSeries xField="Revenue" displayName="Revenue"/>
   </series>
  </Combination2DChart>

type
Type: String

Bar2DSet 클래스 인스턴스에 다중 Bar2DSeries 클래스 인스턴스(시리즈)가 설정된 경우, 시리즈의 표현 방식을 지정합니다.

유효한 값은 아래와 같습니다.
  • "clustered" : 각 시리즈를 나열하듯이 표현합니다. 기본 방식(default)입니다.
  • "overlaid" : 각 시리즈를 덮어씌우는 형식으로 표시하여, 마지막에 표시되는 시리즈가 전면(foreground)에 나타납니다.
  • "stacked" : 각 시리즈를 쌓아올려 표현합니다. 모든 시리즈는 그 아래에 있는 모든 값의 누적치를 나타냅니다.
  • "100%" : 각 시리즈가 100% 대비 비율에 계산되어 쌓이는 방식으로 표현됩니다.
The default value is "clustered".		Bar2DSet
