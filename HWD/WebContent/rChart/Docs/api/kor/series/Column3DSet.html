Class Column3DSetStyles
 
Packagekor.series
Classpublic class Column3DSet
InheritanceColumn3DSet Column2DSet Series

Column3DSet 클래스는 다중의 Column3DSeries 클래스 인스턴스를 하나의 대표 시리즈로 표현합니다.

Column3DSet 으로 대표되는 복수의 Column3DSeries 는 마치 하나의 시리즈와 같이 작동됩니다.

레이아웃은 다음과 같은 형식으로 작성됩니다.

  <Combination3DChart showDataTips="true">
    <horizontalAxis>
     <CategoryAxis categoryField="Month"/>
       </horizontalAxis>
       <series>
          <Column3DSet type="clustered">
            <series>
       <Column3DSeries yField="Cost" displayName="Cost"/>
       <Column3DSeries yField="Revenue" displayName="Revenue"/>
        </series>
       </Column3DSet>
     <Line2DSeries  yField="Revenue" displayName="Revenue"/>
    </series>
   </Combination3DChart>

columnWidthRatio
Type: Number

표시하는 칼럼(column)의 넓이값을 카테고리의 넓이값에 대한 비율(범위는 0 ~ 1)로 지정합니다.
값이 1 이면 영역 전체가 사용되며, 값이 0.6 이면 칼럼(column)의 사용 가능 영역은 60% 가 됩니다.
실제 칼럼(column)의 넓이는 columnWidthRatio 속성과
maxColumnWidth 속성을 계산하여 그중 작은 값이 적용됩니다.
