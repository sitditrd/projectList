Packagekor.series
Classpublic class BoxPlotSeries

BoxPlotSeries 클래스는 BoxPlotChart 클래스 인스턴스의 series 속성에 설정됩니다.

차트에 설정된 데이터를 처리하고 최소, 최대, 제1사분위, 제3사분위 도형을 표현합니다.

레이아웃은 다음과 같은 형식으로 작성됩니다.

  <rMateChart >
   <Options>
    <Caption text="Annual Report"/>
   </Options>
   <BoxPlotChart showDataTips="true">
    <horizontalAxis>
     <CategoryAxis categoryField="Month" />
    </horizontalAxis>
    <series>
     <BoxPlotSeries yField="Profit"/>
     <BoxPlotSeries yField="Revenue" />
    </series>
   </BoxPlotChart>
  </rMateChart>

halfWidthOffset
Type: Number

다중 박스(Box) 시리즈가 존재할 경우, 이웃한 박스(Box) 사이의 간격을 픽셀 단위로 지정합니다.
두 개의 BoxPlotSeries 가 존재하고, halfWidthOffset 속성값이 0.5 이면 두 박스(Box) 사이에는 1px 의 여백이 생깁니다.

The default value is 0.5.		BoxPlotSeries
  
maxColumnWidth
Type: Number

박스(Box)의 최대 넓이값을 픽셀 단위로 지정합니다.
실제 박스(Box)의 넓이는 barWidthRatio 속성과
maxColumnWidth 속성을 계산하여 그중 작은 값이 적용됩니다.

The default value is NaN.		BoxPlotSeries
