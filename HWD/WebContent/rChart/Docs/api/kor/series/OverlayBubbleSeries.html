OverlayBubbleSeries 클래스는 OverlayBubbleChart 클래스 인스턴스의 series 속성에 설정됩니다.

차트에 설정된 데이터를 처리하고 Overlay 버블을 표현합니다.

레이아웃은 다음과 같은 형식으로 작성됩니다.

  <rMateChart ... >
   <OverlayBubbleChart ... >
    <series>
     <OverlayBubbleSeries field="data" displayName="Overlay">
      <showDataEffect>
       <SeriesInterpolate ... />
      <showDataEffect>
     </OverlayBubbleSeries>
    </series>
   </OverlayBubbleChart>
  </rMateChart>
