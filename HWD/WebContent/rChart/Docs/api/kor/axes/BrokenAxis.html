Class BrokenAxisStyles
 
Packagekor.axes
Classpublic class BrokenAxis
InheritanceBrokenAxis LinearAxis NumericAxis AxisBase

BrokenAxis 클래스는 차트에 표현될 일부 데이터 값이 다른 데이터 값들과 현저하게 차이가 날 경우, 다른 스케일 값을 적용할 수 있도록 합니다.

일반적으로 칼럼, 라인 차트의 세로축에 정의됩니다.

레이아웃은 다음과 같은 형식으로 작성됩니다.

   <Column2DChart id="chart1" ...>
    <verticalAxis>
     <BrokenAxis id="hAxis" ... />
    </verticalAxis>
  <Column2DChart>
