Class AxisBaseStyles
 
Packagekor.axes
Classpublic class AxisBase
SubclassesCategoryAxis, NumericAxis

AxisBase 클래스는 LinearAxis, CategoryAxis, DateTimeAxis, LogAxis 클래스들의 기본 클래스입니다.

축 클래스들이 AxisBase 클래스를 상속받기 때문에 사용자는 이를 레이아웃에 설정하지 않아도 됩니다.


Styles
 StyleDescriptionDefined by
  
displayName
Type: String

축의 이름을 지정합니다. 설정된 값은 기본적으로 데이터팁(툴팁)에서 보여지게 됩니다.

AxisBase
  
formatter
Type: Object

축의 라벨값을 원하는 숫자(통화) 형식으로 표현하기 위한 포맷터(NumberFormatter, CurrencyFormatter)를 지정합니다.

AxisBase
  
id
Type: String

식별자(Identifier)를 설정합니다. 식별자(Identifier)는 사용자에 의해 정의된 임의의 고유값입니다.
설정된 식별자(Identifier)를 통해 해당 객체에 접근이 가능합니다.

AxisBase
  
labelJsFunction
Type: String

축 라벨을 사용자 정의 라벨로 표현하기 위한 자바스크립트 함수명을 지정합니다.
차트 레이아웃 설정시 labelJsFunction 속성값을 자바스크립트 함수명으로 지정합니다.

예)
  <horizontalAxis>
   <LinearAxis id="hAxis" labelJsFunction="axisLabelFunc" />
  </horizontalAxis>

labelJsFunction 함수 시그너쳐(signature)는 다음과 같습니다.

  function_name(id, value) { ... }
  id : 축의 식별자(Identifier).
  value : 축의 라벨값.
AxisBase
  
title
Type: String

축의 제목을 지정합니다. 세로축인 경우 세로로 제목이 출력됩니다.
