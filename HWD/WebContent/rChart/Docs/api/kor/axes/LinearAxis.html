Class LinearAxisStyles
 
Packagekor.axes
Classpublic class LinearAxis
InheritanceLinearAxis NumericAxis AxisBase
SubclassesBrokenAxis, CategoryLinearAxis

LinearAxis 클래스는 수치 데이터를 표현하는 축 생성에 적용됩니다.

보통 칼럼, 라인 차트의 세로축에 정의됩니다.

레이아웃은 다음과 같은 형식으로 작성됩니다.

   <Column2DChart id="chart1" ...>
    <horizontalAxis>
     <LinearAxis id="hAxis" ... />
    </horizontalAxis>
  <Column2DChart>
