Package kor.axesClasses
 

The axes package contains the classes for the rMate Chart Axis classes.


Classes
 ClassDescription
 AxisBase AxisBase 클래스는 LinearAxis, CategoryAxis, DateTimeAxis, LogAxis 클래스들의 기본 클래스입니다.
 BrokenAxis BrokenAxis 클래스는 차트에 표현될 일부 데이터 값이 다른 데이터 값들과 현저하게 차이가 날 경우, 다른 스케일 값을 적용할 수 있도록 합니다.
 CategoryAxis 카테고리 축(CategoryAxis)은 숫자가 아닌 문자가 축의 라벨에 표시됩니다.
 CategoryLinearAxis CategoryLinearAxis 클래스는 스크롤 차트의 카테고리 축으로 사용됩니다.
 ColorAxis ColorAxis 클래스는 색상에 대한 범례를 표시할 필요가 있을 경우 사용됩니다.
 DateTimeAxis DateTimeAxis 클래스는 데이터를 시간/날짜 단위로 출력해야 할 경우 사용됩니다.
 LinearAxis LinearAxis 클래스는 수치 데이터를 표현하는 축 생성에 적용됩니다.
 LogAxis LogAxis 클래스는 데이터의 수치값이 커서 로그값으로 축의 값을 표현해야할 필요가 있을 때 적용됩니다.
 NumericAxis NumericAxis 클래스는 LinearAxis, DateTimeAxis, LogAxis 클래스들의 기본 클래스입니다.
