Class ColorAxisStyles
 
Packagekor.axes
Classpublic class ColorAxis

ColorAxis 클래스는 색상에 대한 범례를 표시할 필요가 있을 경우 사용됩니다.

레이아웃은 다음과 같은 형식으로 작성됩니다.

  ...
  <Options>
   <ColorRangeLegend axis="{cAxis}/>
  </Options>
  <TreeMapChart>
   <colorAxis>
    <ColorAxis id="cAxis" minimum="0" maximum="100" colors="[#ffffff, #ff0000]" interval="20"/>
   </colorAxis>
  </TreeMapChart>
  ...
