Packagekor.httpService
Classpublic class HttpServiceRepeater

HttpServiceRepeater 클래스는 지정된 URL 로 RPC 요청을 수행합니다.

RPC 수행에 의해서 서버로부터 받은 데이터를 target 속성에 지정된 객체에 전달합니다.
이때 target 속성에 지정된 객체는 반드시 dataProvider 속성을 갖는 객체여야 합니다.
