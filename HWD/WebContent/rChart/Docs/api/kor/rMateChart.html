알메이트 차트 레이아웃(계층 형태)에서 최상위 노드에 해당됩니다.

생성되는 차트의 배경 및 전체적인 프레임을 설정합니다. 예를 들면, 차트의 배경색, 테두리 스타일, 여백 등을 조절할 수 있습니다.

레이아웃은 다음과 같은 형식으로 작성됩니다.

  <rMateChart  paddingRight="10" paddingBottom="5" paddingTop="5" paddingLeft="10">
    <Options>
     <Caption text="Annual Report"/>
    </Options>
    ...
    ...
    ...
  </rMateChart>

borderStyle
Type: String

차트 테두리 선의 스타일을 지정합니다.

유효값은 none, solid, inset, outset 입니다.
  • none - 테두리 선을 그리지 않도록 설정합니다.
  • solid - 테두리를 기준으로 테두리 자체에 선을 그리도록 설정합니다.
  • inset - 테두리를 기준으로 테두리 안쪽에 선을 그리도록 설정합니다.
  • outset - 테두리를 기준으로 테두리 바깥쪽에 선을 그리도록 설정합니다.
The default value is "none".		rMateChart
