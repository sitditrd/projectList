Class RadialGradientStyles
 
Packagekor.graphics
Classpublic class RadialGradient

RadialGradient 클래스는 원형(Radial) 그라데이션 효과를 지원합니다.

RadialGradient 클래스 인스턴스의 entires 속성에 GradientEntry 클래스 인스턴스 배열을 정의해야 합니다.

레이아웃은 다음과 같은 형식으로 작성됩니다.

  <fill>
    <RadialGradient>
     <entries>
      <GradientEntry color="0xC5C551" ratio="0.00" alpha="0.5"/>
      <GradientEntry color="0xFEFE24" ratio="0.33" alpha="0.5"/>
      <GradientEntry color="0xECEC21" ratio="0.66" alpha="0.5"/>
     </entries>
    </RadialGradient>
  </fill>

게이지 차트에만 지원됩니다.

RadialGradient
  
focalPointRatio
Type: Number

중심에서 그라디언트 초점까지의 거리를 지정합니다.
유효값은 -1 과 1 사이의 값입니다.
예를 들어, 값이 0.25 이면 중심을 기준으로 오른쪽에 초점이 존재하고, -0.25 이면 중심을 기준으로 왼쪽에 초점이 존재합니다.

The default value is 0.		RadialGradient
