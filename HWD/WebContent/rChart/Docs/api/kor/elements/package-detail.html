Package kor.elementsClasses
 

The elements package contains the classes such as AxisLine, AxisMarker, AxisRange, CrossRangeZoomer


Classes
 ClassDescription
 AxisLine AxisLine 클래스는 차트에 표시되는 선에 대한 두께, 색상, 라벨 등과 같은 속성에 대한 설정을 지원합니다.
 AxisMarker AxisMarker 클래스는 차트의 좌표상에 선과 범위(range)에 대한 표시를 지원합니다.
 AxisRange AxisRange 클래스는 차트에 표시되는 영역(range)에 대한 두께, 색상, 라벨 등과 같은 속성에 대한 설정을 지원합니다.
 Box Box 클래스는 Label 과 Image 클래스 인스턴스를 표시할 수 있도록 합니다.
 BrokenAxisLine BrokenAxisLine 클래스는 브로큰 축에 물결 무늬의 표현를 지원합니다.
 CanvasElement CanvasElement 클래스는 Label 과 Image 클래스 인스턴스를 표현할 수 있습니다.
 CrossRangeZoomer CrossRangeZoomer 클래스는 차트상에 마우스 이동에 따른 십자선을 표시하고 특정 영역에 대한 선택과 확대(zoom) 기능을 지원합니다.
 DualScrollBar The DualScrollBar class is set as a class instance in the value of the dataSelector property when the instance of the DualChart class has a lot of data to represent.
 GridLines GridLines 클래스는 차트에서 데이터가 표현되는 영역의 배경에 격자선을 표시합니다.
 Image Image 클래스는 차트의 배경, Box 클래스 인스턴스, CanvasElement 클래스 인스턴스에 이미지의 표시를 지원합니다.
 Label Label 클래스는 차트상에 메모의 표현을 지원합니다.
 RadarGridLines RadarGridLines 클래스는 레이더 차트의 배경에 그리드 선에 대한 표현을 지원합니다.
 SubLegend SubLegend 클래스는 차트에 범례(Legend)에 대한 표현을 지원합니다.
 WingGridLines WingGridLines 클래스는 차트에서 데이터가 표현되는 영역의 배경에 격자선을 표시합니다.
