Class GridLinesStyles
 
Packagekor.elements
Classpublic class GridLines
SubclassesWingGridLines

GridLines 클래스는 차트에서 데이터가 표현되는 영역의 배경에 격자선을 표시합니다.

차트의 축상에 있는 모든 눈금(tick)에 격자선이 표시됩니다. 추가로 차트의 중앙에 세로선과 가로선을 표시할 수 있습니다.

레이아웃에서 backgroundElements 또는 annotationElements 속성값으로 GridLines 클래스 인스턴스를 지정합니다.

레이아웃은 다음과 같은 형식으로 작성됩니다.

  <rMateChart backgroundColor="0xFFFFFF"  borderRadius="12" borderStyle="solid">
  <Bar3DChart>
   <verticalAxis>
            <CategoryAxis categoryField="Month"/>
         </verticalAxis>
   <series>
    <Bar3DSeries xField="Profit">
     <showDataEffect>
        <SeriesInterpolate/> 
       </showDataEffect>
    </Bar3DSeries>
   </series>
   <backgroundElements>
    <GridLines direction="both" verticalChangeCount="1" horizontalChangeCount="1">
        <horizontalStroke>
      <Stroke color="0x999999" alpha="0.5" weight="1"/>
      <!-- color:색깔 alpha:투명도 weight:굵기 -->
     </horizontalStroke> 
     <horizontalFill>
      <SolidColor color="0xEEEEEE" alpha="0.5"/>
      <!-- color:색깔 alpha:투명도 -->
     </horizontalFill> 
     <horizontalAlternateFill>
      <SolidColor color="0xFFFFFF" alpha="0.5"/>
      <!-- color:색깔 alpha:투명도 -->
     </horizontalAlternateFill>
     <verticalStroke>
      <Stroke color="0x999999" alpha="0.5" weight="1"/>
     </verticalStroke>
     <verticalFill>
      <SolidColor color="0xFFFFFF" alpha="0.5"/>
     </verticalFill> 
     <verticalAlternateFill>
      <SolidColor color="0xEEEEEE" alpha="0.5"/>
     </verticalAlternateFill> 
     <horizontalOriginStroke>
      <Stroke color="0x999999" alpha="0" weight="1"/>
     </horizontalOriginStroke>
    </GridLines>
   </backgroundElements>
  </Bar3DChart>
  </rMateChart>


Styles
 StyleDescriptionDefined by
  
direction
Type: String

차트 배경선(gridLine)의 유형을 지정합니다.
유효값은 "vertical"(수직선), "horizontal"(수평선), "both"(수직선과 수평선 동시) 입니다.

The default value is "horizontal".		GridLines
  
horizontalAlternateFill
Type: Object

direction 속성값이 horizontal 일 경우 다음 채우기 색상을 지정합니다. (horizontalFill 속성에 설정된 색상과 번갈아 적용되는 색상.)
유효값은 SolidColor 클래스의 인스턴스입니다.

GridLines
  
horizontalCenterStroke
Type: Object

차트 배경의 중앙에 표시할 가로선의 스타일을 설정합니다.
유효값은 Stroke 클래스의 인스턴스입니다.

GridLines

The default value is 1.		GridLines
