Class RadarGridLinesStyles
 
Packagekor.elements
Classpublic class RadarGridLines

RadarGridLines 클래스는 레이더 차트의 배경에 그리드 선에 대한 표현을 지원합니다.

이 클래스를 사용하기 위해서는 레이아웃에서 backgroundElements 속성값으로 RadarGridLines 클래스 인스턴스를 지정해야합니다.

레이아웃은 다음과 같은 형식으로 작성됩니다.

 <rMateChart backgroundColor="0xFFFFFF" borderRadius="12" borderStyle="solid">
 ...
 <RadarChart id="chart1" isSeriesOnAxis="true" type="polygon" paddingTop="20" paddingBottom="20" showDataTips="true">
  ...
  <backgroundElements>
   <RadarGridLines direction='both' radarChangeCount='1' radarFill='0xEFFFFF' radarAlternateFill='0xFFFFFC'> <!--radarFill 과 radarAlternateFill 색이 차례로 나옵니다.-->
     <radialStroke> <!-- radial 선 정의 -->
     <Stroke color='0x000000' alpha='1' weight='1'/>
    </radialStroke>
     <angularStroke> <!-- angular 선 정의 -->
     <Stroke color='0xFF0000' alpha='1' weight='1'/>
    </angularStroke>
    </RadarGridLines>
  </backgroundElements>
 ...
 </RadarChart>
 </rMateChart>

gridLines
Type: String

레이더 차트의 배경에 표시할 선의 형태를 지정합니다.
값이 radial 이면 중심에서 AngularAxis 인스턴스의 축까지 직선을 표시합니다.
값이 angular 이면 AngularAxis 인스턴스의 축 라벨들을 서로 연결하는 선을 표시합니다.
이때 레이더 차트의 유형(type)이 circle 이면 원으로 선을 표시하고 polygon 이면 다각형으로 선을 표시합니다.

The default value is "both".		RadarGridLines
  
radarAlternateFill
Type: Object

레이더 차트 배경에 대한 다음 채우기 색상을 지정합니다.

The default value is #FCFCFC.		RadarGridLines
  
radarChangeCount
Type: Number

radarFill 인스턴스에 설정된 색상과 radarAlternateFill 인스턴스에 설정된 색상 사이의 간격을 지정합니다.
간격은 radialAxis 속성에 지정된 축의 눈금 간격(tick interval)과 관련이 있습니다.

The default value is 1.		RadarGridLines
