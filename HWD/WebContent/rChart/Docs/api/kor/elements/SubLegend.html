Class SubLegendStyles
 
Packagekor.elements
Classpublic class SubLegend
InheritanceSubLegend Legend

SubLegend 클래스는 차트에 범례(Legend)에 대한 표현을 지원합니다.

차트의 하단에 출력되는 것이 기본(default)이며, 차트를 기준으로 상, 하, 좌, 우에 표시가 가능합니다.
