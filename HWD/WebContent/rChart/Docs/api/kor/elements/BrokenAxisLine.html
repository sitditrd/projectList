Class BrokenAxisLineStyles
 
Packagekor.elements
Classpublic class BrokenAxisLine

BrokenAxisLine 클래스는 브로큰 축에 물결 무늬의 표현를 지원합니다.

물결 무늬를 표현하는 선과 물결 무늬 영역의 색상을 변경할 수 있습니다.

레이아웃은 다음과 같은 형식으로 작성됩니다.

  <annotaionElements>
   <BrokenAxisLine brokenAxisAreaColor="0xffffff" flatRatio="0">
    <brokenAxisLineStroke>
     <Stroke color="0xff0000" weight="1" alpha="1"/>
    </brokenAxisLineStroke>
   </BrokenAxisLine>
  </annotationElements>


Styles
 StyleDescriptionDefined by
  
brokenAxisAreaColor
Type: uint

BrokenAxisLine 클래스의 물결무늬 영역에 대한 색상값을 지정합니다.

The default value is #ffffff.		BrokenAxisLine
  
brokenAxisLineStroke
Type: uint

BrokenAxisLine 클래스의 선에 대한 색상값 지정합니다.

The default value is #454545.		BrokenAxisLine
  
direction
Type: String

BrokenAxisLine 클래스 인스턴스의 출력 방향을 설정합니다.
바((Bar) 차트 계열에는 "vertical"을 그 외에는 기본값(default)을 설정하십시오.
유효값은 "horizontal", "vertical" 입니다.

The default value is "horizontal".		BrokenAxisLine
  
flatRatio
Type: Number

BrokenAxisLine 클래스의 굴곡선의 비율을 지정합니다.
0 값에 가까울수록 굴곡이 강하고, 1 값에 가까울수록 굴곡이 약합니다.

The default value is 0.2.		BrokenAxisLine
