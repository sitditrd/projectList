Class BoxStyles
 
Packagekor.elements
Classpublic class Box

Box 클래스는 Label 과 Image 클래스 인스턴스를 표시할 수 있도록 합니다.

horizontalAlign, verticalAlign 속성에 따라서 박스 내부에 표시될 요소들이 순서대로 표시됩니다.

레이아웃은 다음과 같은 형식으로 작성됩니다.

  <Box horizontalAlign="left" verticalAlign="top">
   <Box direction="vertical">
    <Label text="Test"/>
    <Label text="Test2"/>
   </Box>
   <Box>
    <Column2DChart>
     ...
    </Column2DChart>
   </Box>
  </Box>

borderStyle
Type: String

박스 테두리 선의 스타일을 지정합니다.

유효값은 none, solid, inset, outset 입니다.
  • none - 테두리 선을 그리지 않도록 설정합니다.
  • solid - 테두리를 기준으로 테두리 자체에 선을 그리도록 설정합니다.
  • inset - 테두리를 기준으로 테두리 안쪽에 선을 그리도록 설정합니다.
  • outset - 테두리를 기준으로 테두리 바깥쪽에 선을 그리도록 설정합니다.
The default value is "none".		Box

Type: uint

수평 방향의 스크롤 정책을 설정합니다.

유효값은 아래와 같습니다.
