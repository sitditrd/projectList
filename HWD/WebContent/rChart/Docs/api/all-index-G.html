레이더 차트의 배경에 표시할 선의 형태를 지정합니다.
값이 radial 이면 중심에서 AngularAxis 인스턴스의 축까지 직선을 표시합니다.
값이 angular 이면 AngularAxis 인스턴스의 축 라벨들을 서로 연결하는 선을 표시합니다.
이때 레이더 차트의 유형(type)이 circle 이면 원으로 선을 표시하고 polygon 이면 다각형으로 선을 표시합니다.
