padding — Style in class kor.axes.CategoryAxis

차트의 시리즈 아이템이 출력되는 곳의 여백 크기를 설정합니다.
칼럼, 바 차트 계열의 기본(default) padding 값은 0.5 입니다.
라인, 영역 차트의 padding 값을 0으로 설정한 경우, 차트의 시리즈 아이템이 출력되는 곳의 여백은 없습니다.
여백의 단위는 차트의 시리즈 아이템 하나의 공간이 1 입니다.

placement — Style in class kor.axisRenderers.Axis2DRenderer

차트에서 축이 표시될 위치를 설정하는 속성입니다.
유효한 값은 수직 축인 경우, "left" 와 "right", 수평 축인 경우, "top" 와 "bottom" 입니다.
기본값(default)은 첫번째 축이 왼쪽(수직 축)과 아래(수평 축)이며, 두번째 축이 오른쪽과 위에 배치됩니다.
