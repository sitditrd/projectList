fillsRange — Style in class kor.charts.Equalizer2DChart

fills 속성을 지정할 경우 각 색상이 적용되는 범위를 설정합니다.

fillsRange="[0,20,40,60,80,100]" 와 같이 설정한다면, 색상이 적용되는 구간들은
0~20, 20~40, 40~60, 60~80, 80~100 와 같이 총 다섯개의 구간에 색상이 설정됩니다.
Equalzier2DSeries 의 Y축 값은 fillsRange 속성에 정의된 범위를 기준으로
fillsRange[0] < Y축 값 <= fillsRange[1] 과 같이 계산되어 색상을 설정하도록 합니다.
