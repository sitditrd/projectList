setSelectedIndices — Style in class kor.rMateChartH5

selectionMode가 "single", "multiple" 일 경우 출력되어져 있는 아이템 중 indices 에 맞는 아이템을 강조합니다.

유효값은 아래 형태로 설정합니다.

Array일 경우 -

document.getElementById(chart ID).setSelectedIndices([{index : 0, items : [0, 1, 2]}, ..]); 

  [
   {index : 0, items : [0, 1, 2]},
   {index : 2, items : [2, 3, 4]}
  ]
  • index - 몇 번째 시리즈를 선택할지 index 값
  • items - 몇 번째 아이템들을 선택할지의 배열 값
Object일 경우 -

document.getElementById(chart ID).setSelectedIndices({index : 0, length : 2, items : [0, 1, 2]}); 

  {index : 0, length : 3, items : [0, 1, 2]}
  • index - 몇 번째 시리즈부터 선택할 지 index 값
  • length - index부터 몇 번째 시리즈까지 선택할 지 개수
  • items - 해당 시리즈안에서 선택할 item 들의 배열 값
index가 0 length가 2일 경우 0번째 시리즈부터 시작하여 1번째 시리즈까지 설정합니다.
